Summary

深色聊天页已经切换到正确暗色背景，但暗色玻璃、粉蓝边光和对话气泡质感都不够，整体仍偏向功能实现而不是设计还原。

---------------------------------------

P0（必须修复）

- 暗色玻璃语言不对。
原因：设计稿中的头部、气泡、输入栏都有明显的暗色玻璃、粉紫描边和柔和光晕；当前实现是较平的深色块。
建议修改方式：统一重做暗色聊天页的玻璃底、描边、阴影和发光规则。
涉及组件：Header、AI 气泡、VoiceBubble、Composer。
涉及文件：`web/src/pages/ChatDarkPage.tsx`

- 用户消息气泡错误地使用了平面蓝块。
原因：设计稿中的用户气泡更发光、更柔和，不是纯实色矩形。
建议修改方式：按设计稿重设用户气泡背景、渐变和阴影。
涉及组件：用户消息气泡。
涉及文件：`web/src/pages/ChatDarkPage.tsx`

- 输入栏尺寸与底部悬浮感不足。
原因：设计稿输入区更厚、更贴近底部大胶囊；当前版本像普通表单栏。
建议修改方式：提升容器高度、边缘高光和发送按钮体量。
涉及组件：Composer。
涉及文件：`web/src/pages/ChatDarkPage.tsx`
