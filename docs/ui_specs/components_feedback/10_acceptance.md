## 验收说明

- **像素级验收工具**：建议使用 Figma Dev Mode Overlay 或 PixelSnap 进行尺寸比对

- **无障碍验收工具**：iOS VoiceOver 真机测试，Accessibility Inspector（Xcode）
- **动画验收**：真机运行，使用 iOS Accessibility > Slow Animations 开关辅助检查
