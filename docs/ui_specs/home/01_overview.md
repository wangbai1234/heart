# 01 Overview — 首页 Home

## 页面名称
首页（Home）

## 页面作用
登录后的主入口页面，集中展示：
- 当前绑定 AI 伴侣的情绪宝珠（Heart Orb）与状态
- 三个快速操作功能瓷砖（兑换会员 / 切换角色 / 设置）
- 最近对话角色列表（Recent Conversations）
- 底部浮动 Tab 栏（首页 / 聊天 / 角色 / 设置）

## 页面目标
1. 让用户第一眼感受到 AI 伴侣的"存在感"与情绪温度
2. 提供一键进入对话的主路径（"开始聊天 →"按钮）
3. 快速访问会员、角色切换、系统设置
4. 保持最近对话历史的可见性，降低再次进入对话的摩擦

## 进入方式
- 登录成功后自动跳转
- 点击底部 Tab 栏"首页"图标
- 从任意其他页面点击 Tab 栏"首页"

## 退出方式
- 点击底部 Tab 栏其他标签（聊天 / 角色 / 设置）
- 点击"开始聊天 →"跳转至对话页
- 点击快速操作瓷砖（兑换会员 / 切换角色 / 设置）进入对应页面
- 点击最近对话列表中的角色行，进入对应对话页
- 点击"查看全部 >"进入对话历史列表

## 用户流程
```
登录成功
    ↓
首页（当前页）
    ├─→ 点击"开始聊天" → 与当前伴侣的聊天页
    ├─→ 点击"兑换会员" → 会员购买页
    ├─→ 点击"切换角色" → 角色选择页
    ├─→ 点击"设置"     → 系统设置页
    ├─→ 点击对话行     → 对应角色聊天页
    ├─→ 点击"查看全部" → 对话历史列表页
    └─→ 点击右上角头像 → 个人中心页
```

## 设计理念
"温柔的陪伴感"——用户打开 App 后第一眼看到的不是功能列表，而是一颗发光的心形宝珠和伴侣的名字，营造"ta 在等你"的情感连接。大面积奶油粉背景 + 梦幻云彩横幅 + 玻璃质感组件，构成低干扰、高温暖的视觉氛围。

## 视觉关键词
- **Glassmorphism** — 宝珠卡片、快捷操作瓷砖采用半透明毛玻璃
- **Anime / 二次元** — 角色头像为动漫风格圆形插图，名称字体柔和
- **Healing / 治愈** — 粉紫渐变云彩背景，心形宝珠发光效果
- **Warm / 温暖** — 全局底色为奶油粉 #FFF0F3，樱花粉强调色
- **Dreamy Gradient** — 横幅背景为黄→粉→紫多色渐变云层
- **Soft Glow** — 宝珠有内发光 + 外发光效果，模拟玻璃折射
- **Minimal** — 信息密度克制，主动画元素突出，留白充足
- **Rounded** — 所有卡片/瓷砖/头像/按钮均采用圆角设计

## 情绪传达
| 维度 | 传达内容 |
|------|---------|
| 色彩情绪 | 奶油粉底色 + 樱花粉强调 = 温柔、安全感、少女感 |
| 宝珠形象 | 半透明心形玻璃宝珠 = 情感投射载体，"ta 有心情" |
| 角色状态 | "刚刚和你聊过 · 心情：温柔" = 伴侣有记忆、有情绪 |
| 对话预览 | 角色主动发言片段 = 强化"ta 在等你"的陪伴感 |
| 整体氛围 | Genshin 风格云彩 + 发光宝珠 = 梦幻、治愈、沉浸感 |
