# 02 Layout — 首页 Home

## 画布与设备规格
| 属性 | 值 |
|------|---|
| 设备 | iPhone 14 |
| 逻辑分辨率 | 390 × 844 pt |
| 设计画布 | 1024 × 1536 px |
| 缩放比例 | 1024/390 ≈ 2.626x |
| Safe Area Top | 47 pt（状态栏） |
| Safe Area Bottom | 34 pt（Home Indicator） |

---

## 层级结构总览（Z 轴）

```
z-0  Background Layer       — 全局奶油粉纯色背景
z-1  Page Content           — 主滚动内容区
z-10 Bottom Tab Bar         — 浮动底部导航栏（固定，不随滚动）
z-20 Status Bar             — 系统状态栏（iOS 原生）
```

---


### 6. Bottom Tab Bar（底部 Tab 栏）
| 属性 | 值 |
|------|---|
| 位置 | 固定在页面底部，不随滚动 |
| 高度 | 约 83 pt（含 Safe Area Bottom 34 pt） |

| 布局 | 水平等分，图标+文字垂直居中 |

| 属性 | 值 |

## 滚动区域说明
| 区域 | 是否可滚动 |
|------|-----------|
| Header | 固定，不滚动 |
| Hero Card + 快速操作 + 对话列表 | 可向上滚动（竖向滚动，ScrollView） |
| Bottom Tab Bar | 固定，不滚动 |

设计稿展示为静态非滚动状态，内容区高度约 844 - 47（Safe Top）- 83（Tab Bar）= 714 pt，设计稿中内容基本填满但未产生溢出，推测对话条目若增加则启用滚动。
