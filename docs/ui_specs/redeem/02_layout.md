# 02 Layout — Redeem 兑换页

## 画布规格
- 设计稿画布：1024 × 1536 px
- 逻辑分辨率对应：390 × 844 pt（iPhone 14）
- 缩放比例：约 2.626× (1024/390)
- 以下尺寸均为**设计稿像素值**，换算逻辑像素时除以 2.626

---

## 各区域尺寸

### Status Bar
- 高度：≈ 90px（估算值）
- 背景：透明，背景渐变透出
- 内容：左侧时间「9:41」，右侧信号/WiFi/电量图标

### Navigation Header
- 高度：≈ 96px（估算值）
- 左侧：返回按钮「<」，距左边距约 48px，垂直居中
- 中间：标题「兑换会员」，水平居中，垂直居中
- 右侧：无额外操作图标

### Input Card（主输入卡片）
- 宽度：≈ 944px（左右各留约 40px 外边距）（估算值）
- 高度：≈ 560px（估算值）
- 圆角：≈ 32px（估算值）
- 背景：rgba(255, 248, 243, 0.92)（接近奶油白，轻微半透明）
- 内部 padding：≈ 40px（估算值）
- 内部布局（Flex Column，居中对齐）：
  - 礼品盒插画：≈ 240 × 240px，水平居中（估算值）
  - 插画与标题间距：≈ 32px（估算值）
  - 标题文字「输入兑换码激活会员」：水平居中
  - 标题与副标题间距：≈ 12px（估算值）
  - 副标题文字：水平居中
  - 副标题与输入框间距：≈ 40px（估算值）
  - 兑换码输入框组（4-4-4）：宽度撑满卡片内容区，高度 ≈ 80px（估算值）
  - 输入框与粘贴按钮间距：≈ 32px（估算值）
  - 粘贴按钮：宽度 ≈ 280px，高度 ≈ 72px（估算值），水平居中

### 立即激活 Button
- 宽度：≈ 944px（与卡片同宽，左右同等外边距）（估算值）
- 高度：≈ 96px（估算值）
- 圆角：≈ 48px（完全圆角，pill shape）
- 与上方卡片间距：≈ 40px（估算值）

### FAQ Accordion Card
- 宽度：≈ 944px（估算值）
- 与立即激活按钮间距：≈ 48px（估算值）
- Header 行高度：≈ 80px（估算值）
- 展开状态内容区高度：≈ 460px（估算值，当前截图显示为展开状态）

- 背景：rgba(255, 255, 255, 0.88)（估算值）

- 距 FAQ 卡片底部：≈ 40px（估算值）

### Bottom Safe Area
- 高度：≈ 89px（34pt × 2.626 ≈ 89px）

## 外边距 / 间距体系

| 间距项 | 值（设计稿像素） | 逻辑像素（估算） |
|--------|----------------|---------------|
| 页面左右边距（Horizontal Margin） | ≈ 40px | ≈ 15pt |
| 卡片内 Padding | ≈ 40px | ≈ 15pt |
| 组件间垂直 Gap（主要） | ≈ 40px | ≈ 15pt |
| 标题与副标题 Gap | ≈ 12px | ≈ 5pt |
| 步骤列表行间 Gap | ≈ 32px | ≈ 12pt |

> 注：设计稿使用 8pt 基础网格，以 8 的倍数为主要 Spacing 单位。

---

## 滚动说明
- 整个页面内容高度在标准 iPhone 14 视口下可能超出一屏（当 FAQ 展开时）
- 滚动容器：整页垂直滚动
- 状态栏与导航栏：不随页面滚动，固定在顶部
- 立即激活按钮：**不固定**，随页面内容滚动（设计稿中为文档流内）

---

## Flex / Alignment 说明
- 页面整体：Flex Column，alignItems: center
- Input Card 内部：Flex Column，alignItems: center，gap 见上表
- 输入框组（4-4-4）：Flex Row，gap ≈ 16px（估算值），alignItems: center，包含两个「-」分隔符
- FAQ 步骤列表：Flex Column，各行为 Flex Row（badge + 文字）
