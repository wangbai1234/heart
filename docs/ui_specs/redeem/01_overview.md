# 01 Overview — Redeem 兑换页

## 页面名称
兑换会员 (Redeem Membership)

## 页面作用
让用户输入从「爱发电」赞助平台获得的 12 位兑换码，一键激活 yuoyuo 会员资格。

## 目标
1. 降低兑换流程的认知摩擦，让用户在 30 秒内完成激活
2. 向用户传达"礼物"的惊喜感与会员价值
3. 提供清晰的「如何获取兑换码」引导，帮助还未购买的用户完成转化

## 进入方式
- 从「我的」页面 → 会员/订阅入口 → 选择"使用兑换码"
- 从「会员中心」页面 → 兑换码入口
- 从系统通知/站内信深链直达

## 退出方式
- 点击左上角「<」返回上一页
- 激活成功后自动跳转至会员权益页或首页
- 激活失败时停留在本页，Toast 提示错误原因

## 用户流程
```
进入页面
  ↓
看到礼品插画 + 标题（引导情绪）
  ↓
在 4-4-4 分组输入框中逐字符输入兑换码
  OR 点击「粘贴兑换码」一键填入
  ↓
点击「立即激活」
  ↓
[成功] → 跳转会员权益页 / Toast「激活成功！」
[失败] → Toast 错误提示 / 输入框高亮红色边框
```

## 设计理念
- **礼物感**：顶部插画以精美礼品盒为主视觉，强化会员激活的"惊喜礼物"隐喻
- **清晰分组**：兑换码采用 4-4-4 三段式分组输入，降低 12 位长码的输入压力
- **粘贴友好**：单独的「粘贴兑换码」按钮契合用户从邮件复制兑换码的真实场景
- **渐进引导**：折叠式「如何获取兑换码」卡片，对已有码的用户不造成视觉干扰
- **品牌一致**：全页采用 cherry pink 渐变，与 yuoyuo 整体视觉 DNA 保持一致

## 视觉关键词
- Glassmorphism（输入框区域卡片的白色半透明质感）
- Anime（礼品盒插画的二次元风格，柔和高光与丝带细节）
- Warm（整体 cherry pink → 奶油白的暖色渐变背景）
- Soft（圆角极大，无硬边；阴影轻柔扩散）
- Healing（插画配色以粉紫为主，治愈属性）
- Minimal（页面元素精炼，无多余信息干扰）
- Gift / Celebration（礼品盒主题）

## 情绪传达
页面整体传递"惊喜 × 期待 × 温暖"——用户打开此页时感受到的是"我要解锁一份礼物"的愉悦感，而非枯燥的表单填写。粉紫天空背景与礼品盒插画共同营造出轻盈、节日感的氛围，降低用户对"付费激活"的抗拒心理。
