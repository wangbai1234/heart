### 2. 主内容区域（Color Grid Section）

#### 整体

| 属性 | 值 |
|------|---|
| 区域顶部 | 约 120 px（估算值） |
| 区域高度 | 约 580 px（估算值） |
| 横向内边距（左右） | 约 60 px（估算值） |
| 列间距（Gap） | 约 40–48 px（估算值） |
| 布局方式 | Flex Row / 5列等宽 |

#### 各色柱列宽

- 5 列等宽分布，总可用宽度约 1416 px
- 每列宽约 240–260 px（估算值），前4列相近，语义色列略宽
- 主色/辅色/点缀色/表面色各列：约 240 px（估算值）
- 语义色列：约 260–280 px（估算值）

#### 色柱列内部结构

每列包含：
- **列标题行**：色系名称 + 主品牌色 HEX 值
- **9 个色阶行**（50 / 100 / 200 / 300 / 400 / 500 / 600 / 700 / 800 / 900）

每个色阶行的结构（左 → 右）：
```
[色阶数字标签]  [圆角色块]  [HEX 数值]
```

| 元素 | 尺寸 / 属性 |
|------|-----------|
| 色阶标签文字 | 约 16–18 px，字重 Regular，颜色 `#3A3A4A`（估算值） |
| 色块宽度 | 约 120–130 px（估算值） |
| 色块高度 | 约 36–40 px（估算值） |
| 色块圆角 | 约 12–16 px（估算值，pill形/全圆角） |
| HEX 文字 | 约 14–16 px，Monospace 或 SF Pro（估算值） |
| 行间距（Gap） | 约 10–12 px（估算值） |

**特殊标记**：400 色阶（品牌主色）在主色/辅色/点缀色三列中有**高亮徽标**，  
徽标为圆形 pill，内含"400"字样，背景色为对应品牌色（例如主色 `#FFB7C5`），  
文字为白色，作为"BASE"标识。

---

### 3. 语义色区域（Semantic Colors）

位于画布右侧第 5 列，垂直排列 4 个语义卡片：

| 状态 | 背景色（估算） | 图标颜色 | 文字内容 |
|------|------------|---------|---------|
| 成功 / Success | `#B6E2C7`（浅绿） | 绿色圆形图标（✓） | 成功 / Success + `#B6E2C7` |
| 警告 / Warning | `#FFD3A5`（浅橙） | 橙色圆形图标（!） | 警告 / Warning + `#FFD3A5` |
| 错误 / Error | `#F4A6A6`（浅红） | 红色圆形图标（✕） | 错误 / Error + `#F4A6A6` |
| 信息 / Info | `#B6C7F4`（浅蓝紫） | 蓝色圆形图标（i） | 信息 / Info + `#B6C7F4` |

每个语义卡片：
- 宽度：约 220–240 px（估算值）
- 高度：约 70–80 px（估算值）
- 圆角：约 16 px（估算值）
- 内部布局：Icon 左对齐 + 文字右侧双行（名称 + HEX）
- 卡片间距：约 12–16 px（估算值）

---

4 个玻璃卡片并排，等宽等间距分布：

| 属性 | 值 |
|------|---|
| 每卡片宽度 | 约 310–330 px（估算值） |
| 每卡片高度 | 约 160–170 px（估算值） |

## 组件层级（Z-index 概念）

| 层级 | 内容 |
|------|------|
| 最底层（z=0） | 画布背景色 `#FFF8F3` |
| 内容层（z=1） | 色柱、色阶行、语义卡片 |
| 玻璃层（z=1） | 玻璃叠加卡片（含底层渐变图） |
| 徽标层（z=2） | 400 色阶的高亮圆形标签 |
| 文字层（z=3） | 所有标题、标签、HEX 文字 |

## 对齐规则

| 规则 | 说明 |
|------|------|
| 主标题 | 画布水平居中 |
| 5个色柱 | Flex Row，justify-content: space-between |
| 每色柱内色阶行 | Flex Column，align-items: flex-start |
| 色阶行内元素 | Flex Row，align-items: center，gap 固定 |
| 语义卡片 | Flex Column，统一宽度 |
| 玻璃卡片 | Flex Row，均等间距 |
| 页脚 | 水平居中，Flex Row，gap 约 8 px |
