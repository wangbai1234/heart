# 01 概览（Overview）— 颜色系统色板

## 页面 / 资产名称

**yuoyuo · 颜色系统**（Color Palette / 色板）

---

## 资产作用

该画布是 yuoyuo AI 陪伴应用的**颜色设计基础规范**，定义全产品色彩体系。  
作用：
- 为所有 UI 组件提供统一色彩参考源（Single Source of Truth）
- 规范 5 个色系的完整 9 级色阶（50–900）
- 定义 4 种玻璃叠加（Glass Overlay）透明度，用于磨砂玻璃表面
- 定义 4 种语义色（成功/警告/错误/信息），配套图标和背景色

---

## 目标

1. 让设计师与开发者对颜色取值无歧义
2. 覆盖全产品所有场景的颜色需求（文本/背景/状态/玻璃层）
3. 为 Tailwind / CSS 变量 / iOS SwiftUI Color 提供直接映射基础
4. 传达品牌调性：温柔、治愈、梦幻、二次元

---

## 设计理念

### 核心方向

- **60% WeChat 清晰度 + 40% Genshin 温暖感**
- 以**粉樱（Cherry Blossom Pink）** 为主色，传递柔软与亲近
- 以**梦幻天蓝（Sky Blue）** 为辅色，代表轻盈与科技感
- 以**薰衣草（Lavender Purple）** 为点缀，赋予神秘与疗愈属性
- **暖奶油 → 炭灰** 的表面中性色系覆盖所有背景与文字层级

### 色彩层级哲学

| 层级 | 色阶范围 | 典型使用场景 |
|------|---------|------------|
| 超浅 | 50–100 | 页面背景、大面积底色 |
| 浅色 | 200–300 | 卡片背景、Tag 底色 |
| 标准（品牌色） | 400 | 主 CTA、品牌标识、重点 icon |
| 中深 | 500–600 | Hover 状态、次级按钮 |
| 深色 | 700–800 | Active 按压态、文字强调 |
| 极深 | 900 | 深色模式基底、最深文字 |

### 玻璃体系哲学

磨砂玻璃叠加（Glass Overlay）使用纯白色以不同透明度叠加，  
35% → 55% → 75% → 90% 的四档覆盖了从**隐约透明**到**接近不透明**的完整范围，  
配合底层渐变背景产生**真实的玻璃质感**。

## 视觉关键词

- **Glassmorphism** — 半透明磨砂玻璃层，模糊背景透出底层色彩
- **Anime** — 配色受日系动画美学影响，粉、蓝、紫三色并置
- **Warm** — 暖奶油底色作为全局基底，不刺眼不冷漠
- **Soft** — 所有色阶均为低饱和度柔和变体，避免强对比冲突
- **Healing** — 颜色组合产生视觉放松与情绪安抚感
- **Minimal** — 仅 5 个主要色系，无多余颜色，系统高度克制
