## A01 — 画布暖奶油底色（背景）

| 属性 | 值 |
|------|---|
| 类型 | 纯色背景 |
| 颜色 | `#FFF8F3`（color-surface-50） |

| 建议命名 | `deco-glass-glow` |

## 资源命名规范

```
类型前缀：
- bg-    背景相关
- icon-  图标相关
- deco-  纯装饰性资源
- brand- 品牌标识相关
- gradient- 渐变相关

语义后缀：
- -primary / -secondary / -accent  色系相关
- -success / -warning / -error / -info  语义状态相关
- -glass  玻璃效果相关
- -brand  品牌专用
```

## 资源使用注意事项

1. 所有语义图标（A03–A06）建议使用 SVG 格式，以支持颜色主题切换
2. 玻璃渐变背景（A02/A10）建议使用 CSS Gradient 代码实现，不用图片资源
3. 品牌文字（A07/A08）应确保字体加载完成后再显示，建议配置字体 fallback
4. 爱心图标（A09）若使用 emoji，需注意跨平台渲染差异，建议使用 SVG 版本
