# 02 Layout — 聊天页 Chat（深色模式）

## 画布与设备规格
- 画布尺寸：1024 × 1536 px（设计稿）
- 逻辑尺寸参考：390 × 844 pt（iPhone 14）
- Safe Area Top：47 px（逻辑）/ ~122 px（设计稿估算）
- Safe Area Bottom：34 px（逻辑）/ ~88 px（设计稿估算）
- 所有尺寸以设计稿像素（px）标注，括号内为逻辑pt估算值

---

## 层级结构（Z-index 从底到顶）

| 层级 | 名称 | Z-index | 说明 |
|------|------|---------|------|
| 0 | Background Layer | 0 | 星云渐变背景图/纹理 |
| 1 | Message List (Scroll) | 1 | 可滚动消息区域 |
| 2 | Header Bar | 10 | 毛玻璃导航栏，fixed top |
| 3 | Input Bar | 10 | 毛玻璃输入栏，fixed bottom |
| 4 | Status Bar | 20 | 系统状态栏（时间/信号） |

---

## 各区域尺寸规范

### 1. 状态栏 Status Bar
- 高度：~88 px（估算值）/ 47 pt 逻辑
- 宽度：1024 px 全宽
- 内容：左侧时间"9:41"，右侧信号/WiFi/电池图标
- 背景：透明（透出页面背景）
- 内边距：左 ~52 px，右 ~52 px（估算值）

### 2. 顶部导航栏 Header Bar
- 起始Y：状态栏底部 ~88 px
- 高度：~140 px（估算值）
- 宽度：1024 px 全宽
- 背景：毛玻璃，深色 rgba(30,27,40,0.75)，blur ~20 px
- 边框：底部无明显分隔线，底部圆角 ~32 px
- 内边距：左右各 ~40 px，上下各 ~20 px（估算值）
- 布局：Flex Row，AlignItems: center，SpaceBetween
- 子元素：返回按钮（左）| 头像+名称+状态（中左）| 更多按钮（右）

  #### 2.1 返回按钮 Back Button
  - 尺寸：约 60 × 60 px 触摸区域（估算值）
  - 图标："<"箭头，约 24 px 高，颜色 #FFFFFF / #E0E0F0
  - 位置：Header 左侧，垂直居中

  #### 2.2 头像 Avatar
  - 尺寸：约 100 × 100 px（估算值）/ 圆形
  - 位置：返回按钮右侧，间距 ~16 px
  - 边框：无明显边框，有柔和光晕

  #### 2.3 角色信息区 Name + Status
  - 位置：头像右侧，Flex Column
  - 名称"小屿"：字号 ~32 px（估算值），字重 Medium，颜色 #FFFFFF
  - 状态"温柔在线"：字号 ~22 px（估算值），颜色淡紫/白，前置蓝色圆点 ~10 px
  - 两行间距：~6 px（估算值）

  #### 2.4 更多按钮 More Button
  - 触摸区域：约 60 × 60 px（估算值）
  - 图标："···"三点，颜色 #FFFFFF
  - 位置：Header 右侧，垂直居中

### 3. 时间戳 Timestamp
- 位置：Header Bar 底部与第一条消息之间
- 内容："今天 · 上午 9:41"
- 字号：~22 px（估算值），颜色 rgba(255,255,255,0.45)
- 对齐：水平居中
- 上下边距：上 ~32 px，下 ~24 px（估算值）

### 4. 消息列表区域 Message List（可滚动）
- 起始Y：Header Bar 底部 ~228 px
- 结束Y：Input Bar 顶部 ~1380 px
- 高度：~1152 px（估算值）
- 宽度：1024 px 全宽
- 内边距：左右各 ~32 px（估算值）
- 滚动方向：垂直，内容从上到下，最新消息在底部
- 消息间距（Gap）：~24 px（估算值）
- 溢出：scroll（内容超出时可上下滑动）

  #### 4.1 AI 消息气泡（左对齐）
  - 对齐：左侧，靠近左边距
  - 最大宽度：~680 px（约屏幕宽度 66%，估算值）
  - 内边距：~24 px 上下，~32 px 左右（估算值）
  - 外边距右侧：自动（不超过最大宽度）
  - 气泡类型 A（纯文字）：如"早上好，昨晚睡得怎么样？"
  - 气泡类型 B（语音播放）：含播放按钮 + 波形 + 时长

  #### 4.2 用户消息气泡（右对齐）
  - 对齐：右侧，靠近右边距
  - 最大宽度：~580 px（约屏幕宽度 56%，估算值）
  - 内边距：~24 px 上下，~32 px 左右（估算值）

  #### 4.3 AI "输入中"气泡
  - 对齐：左侧
  - 内容："···"三个粉色圆点动效
  - 尺寸：宽 ~160 px，高 ~80 px（估算值）

### 5. 底部输入栏 Input Bar
- 起始Y：~1380 px（估算值）
- 高度：~120 px（估算值）
- 宽度：1024 px 全宽
- 背景：毛玻璃，深色 rgba(30,27,40,0.80)，blur ~20 px
- 内边距：左右 ~24 px，上下 ~16 px（估算值）
- 布局：Flex Row，AlignItems: center，Gap ~16 px

  #### 5.1 "+"按钮
  - 尺寸：约 80 × 80 px（估算值）/ 圆形
  - 背景：rgba(255,255,255,0.12)
  - 图标："+"加号，颜色 #FFFFFF，约 28 px（估算值）

  #### 5.2 文字输入框
  - 宽度：flex: 1（撑满剩余空间）
  - 高度：约 80 px（估算值）
  - 背景：rgba(255,255,255,0.08)
  - 圆角：~40 px（全圆角，估算值）
  - 占位文字："想和小屿说点什么..." ，颜色 rgba(255,255,255,0.40)
  - 字号：约 28 px（估算值）
  - 内边距：左 ~24 px，右 ~24 px（估算值）

  #### 5.3 发送按钮
  - 尺寸：约 84 × 84 px（估算值）/ 圆形
  - 背景：#FFB7C5（樱花粉）或渐变粉色
  - 图标：纸飞机/发送箭头，颜色 #FFFFFF，约 32 px（估算值）

### 6. Home Indicator（系统底部指示条）
- 高度：~5 px 线条
- 颜色：rgba(255,255,255,0.30)
- 位置：底部 Safe Area 内，水平居中
- 宽度：约 280 px（估算值）

---

## Padding / Margin / Gap 汇总

| 区域 | 类型 | 值（估算值）|
|------|------|------------|
| Header 左右内边距 | Padding Horizontal | ~40 px |
| 消息列表左右内边距 | Padding Horizontal | ~32 px |
| 消息间距 | Gap (Vertical) | ~24 px |
| AI 气泡内边距 | Padding | ~24 px 上下，~32 px 左右 |
| 用户气泡内边距 | Padding | ~24 px 上下，~32 px 左右 |
| Input Bar 内边距 | Padding Horizontal | ~24 px |
| Input Bar 元素间距 | Gap | ~16 px |

---

## 对齐规则
- AI 消息：Flex start（左对齐）
- 用户消息：Flex end（右对齐）
- 时间戳：Center（水平居中）
- Header 内容：Space-between（两端对齐）
- Input Bar 内容：Center（垂直居中）

---

## 滚动区域
- 消息列表为唯一可滚动区域
- 滚动方向：垂直
- Header 和 Input Bar 固定（position: fixed 或 sticky）
- 新消息出现时自动滚动至底部
