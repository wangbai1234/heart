# 10 验收 Checklist — Shadow / Radius / Glass Sample

> Pixel Perfect 验收标准，目标整体视觉一致度 ≥ 95%。
> 本清单用于研发还原后的视觉走查，以及设计师的交付验收。

---

## 使用说明

- `□` — 待检查
- `✅` — 通过
- `❌` — 不通过，需修复
- `（估算值）` — 测量时允许 ±2px 误差

---

## A. 画布与整体布局

```
□ 画布尺寸为 1024 × 1024 px（参考图不适用，实际还原为 App 内嵌时按设备尺寸）
□ 页面背景为奶油白 #FFF8F3，右上角有明显樱花粉光晕
□ 光晕色调为浅粉（不偏红，不过于鲜艳）
□ 主容器卡片居中，左右边距约 42px（估算值），允许 ±4px 误差
□ 主容器卡片顶部边距约 40px（估算值）
□ 主容器底部边距约 40px（估算值）
□ 主容器卡片背景为白色（非纯灰，不偏黄）
□ 主容器卡片有可见的柔和阴影（shadow.sheet 等级，模糊约 24px）
□ 主容器卡片圆角约 32px（视觉上明显圆润，非直角）
```

---

## B. 页面标题

```
□ 标题文字为 "yuoyuo · 设计令牌参考"，yuoyuo 全小写
□ "·" 为中文居中点，前后有空格
□ 字号约 28–32px（估算值），字重 SemiBold
□ 颜色为 #3A3A4A（炭黑，非纯黑 #000000）
□ 标题左对齐
□ 标题与第一个 Section 之间有约 32px 间距（估算值）
```

---

## C. Section 标题行（通用，3 个 Section 均需检查）

```
□ 每个 Section 标题左侧有一个实心圆点图标
□ 圆点颜色为 #FFB7C5（樱花粉，非红色，非橙色）
□ 圆点直径约 12px（估算值）
□ 圆点与标题文字 Gap 约 8px（估算值）
□ 标题字号约 18–20px（估算值），字重 Medium
□ 标题颜色为 #3A3A4A
□ 标题与下方内容之间有约 24px 间距（估算值）
□ Section 1（圆角）标题文字：圆角半径（Radius）
□ Section 2（阴影）标题文字：高度（Elevation）
□ Section 3（玻璃）标题文字：玻璃效果（Glassmorphism）
□ 三个圆点颜色完全一致（同为 #FFB7C5）
```

---

## D. Section 1 — 圆角半径

```
□ 共 5 个示例方块，横排均匀分布
□ 方块间距约 16px（估算值）
□ 方块尺寸约 140 × 140 px（估算值），允许 ±8px 误差
□ 方块背景为白色
□ 方块有轻微阴影（shadow.card 等级，视觉上比主容器阴影更浅）

□ 第 1 个方块圆角为 4px — 视觉上几乎是直角（仅轻微圆角）
□ 第 2 个方块圆角为 8px — 轻微圆角，明显大于 4px
□ 第 3 个方块圆角为 16px — 中等圆角，占方块宽约 11%
□ 第 4 个方块圆角为 24px — 大圆角，视觉上接近卡片感
□ 第 5 个方块圆角为 32px — 最大，视觉上几乎是胶囊/大圆角矩形

□ 5 个圆角值从左到右递增（4 → 8 → 16 → 24 → 32），视觉渐变明显
□ 每个方块正下方有标签文字（"4px"、"8px"、"16px"、"24px"、"32px"）
□ 标签文字居中对齐，与方块底部间距约 12px（估算值）
□ 标签字号约 14px，颜色 #3A3A4A
```

---

## E. Section 2 — 高度（阴影）

```
□ 共 5 个示例方块，横排均匀分布
□ 方块间距约 16px（估算值）
□ 方块尺寸约 140 × 140 px（估算值）
□ 所有方块圆角统一约 16px（radius.md）
□ 方块背景为白色

□ 第 1 个方块（平面）：无阴影，与背景几乎融合（需仔细观察边缘）
□ 第 2 个方块（柔和）：有极轻阴影，blur 约 6px，几乎感觉不到但存在
□ 第 3 个方块（卡片）：明显卡片浮起感，blur 约 12px
□ 第 4 个方块（薄片）：较强浮起感，blur 约 24px，阴影扩散范围更大
□ 第 5 个方块（模态）：最强浮起感，blur 约 40px，阴影柔和但面积最大

□ 5 个方块阴影强度从左到右明显递增
□ 每个方块下方有主标签（名称 + 参数）
□ 主标签格式：第 1 个"平面 (0)"，第 2 个"柔和 (6/0.04)"，以此类推
□ 每个方块下方有副描述（参数说明）
□ 副描述第 2 个："阴影模糊 6px / 不透明度 0.04"
□ 副描述第 3 个："阴影模糊 12px / 不透明度 0.06"
□ 副描述第 4 个："阴影模糊 24px / 不透明度 0.08"
□ 副描述第 5 个："阴影模糊 40px / 不透明度 0.10"
□ 副描述字号约 12px，颜色比主标签更淡（约 60% 不透明度）
□ 主标签和副描述均居中对齐
```

---

## F. Section 3 — 玻璃效果

```
□ 插画背景容器圆角约 20px（估算值）
□ 插画背景有圆角裁切（overflow:hidden）
□ 插画内容：蓝天 + 白云 + 樱花树 + 飘落花瓣（二次元风格）
□ 插画整体色调柔和，非饱和艳丽风格

□ 共 4 个玻璃卡片叠在插画背景上
□ 玻璃卡片横排均匀分布
□ 玻璃卡片间距约 16px（估算值）
□ 玻璃卡片尺寸约 180 × 140 px（估算值）
□ 玻璃卡片圆角约 16px（radius.md）

□ 第 1 个玻璃卡片（35%）：最通透，背景插画透过率最高，轻薄感
□ 第 2 个玻璃卡片（55%）：中等遮蔽，标准玻璃感
□ 第 3 个玻璃卡片（75%）：较不透明，接近半实体
□ 第 4 个玻璃卡片（着色）：有轻微色调叠加（浅粉或浅蓝），区别于前 3 个

□ 所有玻璃卡片均有可见的背景模糊效果（背景插画在卡片后方模糊处理）
□ 卡片与卡片之间的背景插画清晰，对比明显
□ 背景模糊程度统一（4个变体模糊程度相同，约 blur 16px）

□ 每个玻璃卡片下方有 3 行标签文字
□ 第 1 个："玻璃 35%" / "白色 35%" / "背景模糊 16px"
□ 第 2 个："玻璃 55%" / "白色 55%" / "背景模糊 16px"
□ 第 3 个："玻璃 75%" / "白色 75%" / "背景模糊 16px"
□ 第 4 个："玻璃着色" / "白色 55% + 色调" / "背景模糊 16px"
□ 标签居中对齐，字号主标签约 14px，副描述约 12px
```

## G. 底部署名行

```
□ 底部署名内容："♥ yuoyuo · 温柔治愈的陪伴体验"
□ ♥ 图标颜色为 #FFB7C5（樱花粉）
□ "yuoyuo" 全小写，字体为 SF Pro Rounded / PingFang SC
□ 文字颜色为 #3A3A4A 或接近
□ 署名整体水平居中
□ 署名与主容器底部或上方内容有约 24px 间距（估算值）
□ 字号约 13–14px
```

---

## H. 间距一致性

```
□ 三个 Section 之间的垂直间距约 40px（估算值），视觉上均等
□ 主容器左右内 Padding 约 48px（估算值），方块组不会碰到容器边缘
□ 方块下方标签与下一行元素间距足够（无文字压缩）
□ Section 标题与方块组之间有明确间距（约 24px，估算值）
```

---

## I. 视觉权重与层级

```
□ 主容器卡片整体比背景"浮起"感明显
□ 阴影方块组中，5 个方块的层级差异肉眼可区分（从左到右越来越"重"）
□ 玻璃卡片在插画背景上有明显的层叠感（非平面化）
□ 页面标题视觉权重最重（最大、最粗）
□ Section 标题次之
□ 方块标签再次之
□ 副描述最轻（最小、最淡）
```

---

## J. 品牌一致性

```
□ 产品名 "yuoyuo" 全程小写，无任何大写变体
□ 品牌粉 #FFB7C5 只用于装饰色（圆点、♥），不用于正文文字
□ 整体色调偏暖（奶油白底色），不偏冷
□ 玻璃卡片有轻微边框光泽（可选，若设计有），不是纯平的遮罩矩形
□ 字体为 PingFang SC / HarmonyOS Sans SC（中文），SF Pro Rounded（英文/数字）
□ 插画风格为二次元治愈风格，不是写实风格，不是扁平插画风格
```

---

## K. 整体一致度目标

```
□ 圆角 Token（5级）还原准确度 ≥ 95%（4/8/16/24/32px 完全精确）
□ 阴影 Token（5级）还原准确度 ≥ 95%（模糊值精确，不透明度精确）
□ 玻璃遮罩透明度（35%/55%/75%/着色）准确度 ≥ 95%
□ 玻璃背景模糊 16px 精确
□ 整体视觉一致度（设计稿 vs 还原）≥ 95%
```
