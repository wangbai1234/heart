## 10 玻璃卡片（GlassCard）

| 属性 | 值 |
|------|-----|
| 组件名称 | `GlassCard` |
| 作用 | yuoyuo 核心复用组件，叠于渐变/插画背景上的半透明卡片 |
| 层级 | 相对于 IllustrationBackdrop Z-index +1 |
| 尺寸 | 约 180 × 140 px（估算值） |
| 圆角 | 约 16px（radius.md） |
| 背景 | `rgba(255,255,255, opacity)` — 见变体 |
| 背景模糊 | `backdrop-filter: blur(16px)` |
| 边框 | 极淡白色边框 rgba(255,255,255,0.6)（估算值） |
| 交互 | 无（参考图）；实际使用中可有 Pressed 状态 |
| 状态 | Default（4变体展示）；Pressed / Hover 设计稿未定义 |
| **可复用性** | ✅ 是 — 整个 App 的核心 Surface 组件 |

### 4种变体规格

| 变体名 | 白色遮罩不透明度 | 背景模糊 | 额外样式 |
|--------|---------------|---------|---------|
| `glass-35` | 0.35 | blur(16px) | 无 |
| `glass-55` | 0.55 | blur(16px) | 无 |
| `glass-75` | 0.75 | blur(16px) | 无 |
| `glass-tinted` | 0.55 | blur(16px) | + 色调叠加（浅粉或浅蓝，估算值） |

### Props 建议（规格记录，非代码）

- `opacity`: 0.35 | 0.55 | 0.75
- `tinted`: boolean
- `tintColor`: string（可选，默认无色调）
- `radius`: 来自 radius Token（默认 radius.md = 16px）
- `blurAmount`: number（默认 16px）
