# 05 资源清单 — Shadow / Radius / Glass Sample

本模块是设计令牌参考，不是业务页面资源清单。本文件只保留对基础视觉样式的说明，不再引用不存在的演示插画文件。

## 1. 模块定位

- `elevation_radius_glass` 用于定义阴影、圆角、毛玻璃的表现原则
- 它不是一个需要独立实现的业务页面
- 不需要也不允许为此生成专用背景插画资源

## 2. 允许的资源类型

- 纯代码渐变
- 纯代码圆点
- 字体渲染文字
- 现有 token

## 3. 不再允许的旧资源口径

以下旧资源名不再作为执行依据：

- `bg_page_cherry_glow.png`
- `illus_sakura_sky_bg.png`
- `ic_heart_brand.svg`

原因：

- 它们不是业务实现必须资源
- 本地资源目录中没有这些文件
- 容易误导 AI 自行补图

## 4. 正确实现方式

- 背景氛围：使用代码渐变
- Section 装饰圆点：使用纯 CSS / SVG
- 品牌文字：字体渲染
- 毛玻璃样本：通过组件和 token 实现

## 5. 禁止事项

- 禁止补画樱花背景插图
- 禁止为了演示页单独产出一组 PNG
- 禁止把设计令牌参考页误实现为业务页面
