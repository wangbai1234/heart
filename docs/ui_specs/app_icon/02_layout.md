## 对齐与间距原则

- 两版本图标**垂直居中对齐**，视觉重心约在画布 y=470（估算值）
- 两版本尺寸**基本等大**，单色版略小约 5%（估算值），避免等大时视觉上单色版偏大的错觉
- 两版本与画布边缘各留约 55–80 px 的透气间距（估算值）
- 图标整体**视觉重心偏上**，约在造型高度的 42% 处（估算值），符合二次元 IP 审美的「重上轻下」
