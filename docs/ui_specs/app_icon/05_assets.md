## A-01 图标主体矢量路径

| 属性 | 值（估算值）|
|------|------------|
| **建议文件名** | `icon_shape_path.svg` |
| **格式** | SVG（矢量，可无损缩放） |
| **尺寸** | 画布 1024 × 1024 px，路径主体约 380 × 420 px |
| **位置** | 画布中心对称（左版本中心 ≈ 240,480；右版本 ≈ 770,470） |
| **作用** | 所有图标变体的基础路径，单一 source of truth |
| **透明度** | 1.0（路径本身不透明，填充和描边单独控制） |
| **说明** | 路径应为单一闭合路径（Compound Path），包含心形主体和气泡尾巴；顶部双峰、主体弧线、右下收拢尖点、左下尾巴均在同一路径内 |

## A-02 全色渐变填充图层

| 属性 | 值（估算值）|
|------|------------|
| **建议文件名** | `icon_gradient_fill.svg` 或 Figma 图层 |
| **格式** | SVG 渐变填充 或 Figma Fill |
| **渐变类型** | 线性渐变，135° 方向（左下→右上） |
| **颜色** | `#FFB7C5` → `#C8B6FF` → `#A7C7E7` |
| **透明度** | 1.0 |
| **裁切方式** | 使用 icon_shape_path 作为 Clip Mask |
| **说明** | 渐变色彩应与品牌 Primary / Accent / Secondary 保持一致，任何调色须同步更新 design token |

## 资源命名规范

```
icon_{version}_{size}.{ext}

版本标识：
  full   = 全色渐变版
  mono   = 单色版
  dark   = 深色模式专用版

示例：
  icon_full_1024.png
  icon_mono_1024.png
  icon_full_180.png
```

## 注意事项

1. **全色版导出不含 Alpha 通道**：iOS App Store 要求图标为不透明 PNG，背景须为实色（建议使用 `#FFF8F3` 奶油白或直接合并背景色）
2. **单色版遵循 iOS Tinted Icon 规范**：颜色信息由系统接管，图标应提供「只有形状信息」的版本
3. **小尺寸测试**：60 × 60 px 时须确认云朵纹理和星光仍然可见或优雅降级（若不可见可在小尺寸版本中去除）
4. **圆角遮罩**：iOS 系统会自动应用「超圆角」圆角遮罩（squircle），图标设计无需自行裁切圆角，但应确保关键元素不在距离边缘 10% 范围内
