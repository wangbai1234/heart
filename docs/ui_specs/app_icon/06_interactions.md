# 06 交互规范 — App Icon（多色版 + 单色版）

## 说明

本文件为 App Icon 资源的交互规范。作为 PNG 静态图标，设计稿本身不包含任何交互状态定义。以下所有动效规范均标注来源，分为：

- **系统默认**：iOS / Android 系统自动处理，无需研发实现
- **设计建议**：设计师建议添加，需研发评估实现成本
- **设计稿未定义**：明确标注

---

## 进入动画

### 桌面图标首次出现动画（系统默认）

| 属性 | 说明 |
|------|------|
| 来源 | 系统默认（iOS SpringBoard） |
| 动效类型 | 从模糊到清晰 + 缩放（Scale 0.5 → 1.0） |
| 时长 | 约 400ms（系统控制） |
| 触发条件 | App 安装完成后首次出现在桌面 |
| 研发工作量 | 无需实现 |

### App 启动时图标动画（设计建议）

| 属性 | 说明（估算值）|
|------|------------|
| 来源 | 设计建议 |
| 动效类型 | 图标放大 + 淡出，过渡到 Splash Screen |
| Scale | 1.0 → 1.15（估算值） |
| Opacity | 1.0 → 0.0 |
| 时长 | 300ms（估算值） |
| Easing | `ease-in`（估算值） |
| 研发工作量 | iOS: 使用 UIViewPropertyAnimator 实现 |

## Hover 状态

**设计稿未定义**

移动端（iOS/Android）图标不存在悬停状态。
Web/Desktop 桌面应用如需 hover 状态，建议：
- 轻微放大：Scale 1.0 → 1.05
- 外发光增强：opacity 从 40% → 60%
- 时长：150ms，easing: ease-out

---

## Pressed（按下）状态

### 系统长按（iOS 系统默认）

| 属性 | 说明 |
|------|------|
| 来源 | 系统默认 |
| 动效类型 | 长按触发上下文菜单（Haptic Touch），图标轻微震动 |
| 研发工作量 | 无需实现 |

### 按下视觉反馈（设计建议）

| 属性 | 值（估算值）|
|------|------------|
| 来源 | 设计建议 |
| Scale | 1.0 → 0.94（估算值） |
| 时长（按下） | 100ms（估算值） |
| Easing（按下） | `ease-out` |
| 时长（释放弹回） | 400ms（估算值） |
| 弹性（释放） | Spring 动效，轻微过冲（overshoot ≈ 1.03，估算值） |
| 亮度变化 | 轻微降低 5%（估算值） |
| 适用平台 | iOS（使用 SwiftUI `.buttonStyle` + spring animation） |

| 属性 | 值（估算值）|
|------|------------|
| 来源 | 设计建议 |

## 总结：研发工作量分类

| 交互项 | 来源 | iOS 工作量 | Android 工作量 |
|--------|------|-----------|--------------|
| 桌面出现动画 | 系统 | 无 | 无 |
| 长按菜单 | 系统 | 无 | 无 |
| App 角标 | 系统 | 无（系统处理） | 无 |
| 按下压缩弹性 | 设计建议 | 低（SwiftUI spring） | 低（Compose animation） |
| 启动过渡 Hero 动效 | 设计建议 | 中（自定义转场） | 中 |
| 桌面呼吸光效 | 设计建议 | 不适用 | 高（Live Icon） |
