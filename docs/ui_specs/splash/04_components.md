# 04 组件规范 — Splash Screen（启动屏）

> 本页面为启动屏，组件相对简单。逐一分析所有可见组件，并标注复用建议。

---

## 组件总览

| # | 组件名称 | 中文名 | 层级 | 复用性 |
|---|----------|--------|------|--------|
| 1 | `SplashBackground` | 渐变天空背景 | 底层 | 仅启动屏 |
| 2 | `CloudDecoration` | 云朵装饰 | 背景叠加层 | 低（特定场景）|
| 3 | `TopGlowEffect` | 顶部光晕 | 装饰层 | 低 |

## 组件详细规范

---

### 1. SplashBackground（渐变天空背景）

**作用**：全屏覆盖的渐变背景，营造梦幻天空氛围。

| 属性 | 值 |
|------|-----|
| 层级（z-index） | 0（最底层）|
| 尺寸 | 100vw × 100vh（填满全屏）|
| 渐变方向 | 从上到下（180°）|
| 渐变起止 | 顶部奶油黄 `#FCEACB`（估算值）→ 底部淡紫 `#D9B8E8`（估算值）|
| 渐变节点 | 5节点多彩渐变（见 03_design_tokens.md）|
| 内容 | 纯色渐变（云朵作为独立层）|
| 交互 | 无 |
| 状态 | 单一静态状态 |

**样式特征**：
- 顶部偏奶油暖黄，中段为粉红主调，底部向紫粉过渡
- 整体色调连贯，无明显分段感，呈现大气「天空渐变」效果
- 不使用图片，纯 CSS/Canvas 渐变即可实现

**复用建议**：不建议作为独立组件复用，但渐变 token 可在其他「天空主题」页面使用。

---

### 2. CloudDecoration（云朵装饰）

**作用**：分布在画面四周的云朵装饰，增加背景层次感和插画氛围。

| 属性 | 值 |
|------|-----|
| 层级（z-index） | 1 |
| 数量 | 约 8～12 个云朵实例（估算值）|
| 分布 | 四角及左右两侧边缘，顶部较多，底部偏少 |
| 颜色 | 白色至浅粉（`rgba(255,240,245, 0.7)` 估算值）|
| 透明度 | 0.4～0.75（估算值，远景更透明）|
| 模糊 | 边缘柔化（blur ≈ 8～20px，估算值）|
| 风格 | 二次元插画风云朵，有机曲线，无锐角 |
| 尺寸（单个） | 大云朵 ≈ 200×100px（估算值），小云朵 ≈ 80×50px（估算值）|
| 交互 | 无 |

**可见位置（估算）**：
- 左上角（Y ≈ 200px）：2～3 个云朵组群
- 右侧中部（Y ≈ 700px）：零星小云朵
- 左侧中下部（Y ≈ 1000px）：横向云朵条带
- 右下角（Y ≈ 1200px）：大型云朵装饰
- 底部延伸（Y ≈ 1350px）：宽扁型云朵横向分布

**样式特征**：
- 云朵本身低饱和度，接近白色，与背景融合自然
- 部分云朵带有淡粉色/淡紫色反光（与背景渐变协调）
- 无明显硬边，属于「绘制型」云朵而非几何形云朵

**复用建议**：可以将云朵作为 `<svg>` 或 `<img>` 资产导入，通过 CSS `position: absolute` 灵活布局。不建议做成通用组件（场景高度定制化）。

---

### 3. TopGlowEffect（顶部光晕）

**作用**：右上角的暖白色放射状光晕，模拟阳光照射效果，增加画面温暖感。

| 属性 | 值 |
|------|-----|
| 层级（z-index） | 2 |
| 位置 | 右上角，约 (800, 0)～(1024, 200) 区域（估算值）|
| 类型 | Radial Gradient / 光晕叠加 |
| 颜色 | 暖白 `rgba(255,253,230,0.85)` → 透明（估算值）|
| 半径 | ≈ 300px（估算值）|
| 混合模式 | Screen 或 Normal（估算值）|
| 透明度 | 0.8（估算值）|
| 交互 | 无 |

**复用建议**：不建议复用，作为背景层的组成部分内嵌。

---

### 4. GemIcon（心形宝石图标）

**作用**：启动屏核心视觉，品牌标志性图标。玻璃质感心形，融合「心形」与「对话气泡」造型，象征 AI 陪伴与对话。

| 属性 | 值 |
|------|-----|
| 层级（z-index） | 10 |
| 位置 | 水平居中，垂直居中偏上 |
| 宽度 | ≈ 420px（估算值，≈ 160pt） |
| 高度 | ≈ 400px（估算值，≈ 152pt） |
| 形状 | 心形 + 底部类气泡尾部（右下侧小延伸）|
| 风格 | 3D 拟物玻璃质感（Glassmorphism 3D）|

**颜色/渐变**：

| 区域 | 颜色 |
|------|------|
| 左侧主体 | 粉红 `#FFB7C5` 为主 |
| 右侧主体 | 紫蓝 `#C8B6FF` → `#A7C7E7` 渐变 |
| 左上高光 | 白色反光 `rgba(255,255,255,0.9)` |
| 内部光晕 | 浅白/浅紫色散射光（星形粒子点缀）|
| 边缘描边 | 细白描边 `rgba(255,255,255,0.6)`（估算值）|
| 整体轮廓 | 柔和发光（drop shadow，紫粉色）|

**材质层次（从里到外）**：
1. 基础色填充（粉→紫蓝渐变）
2. 玻璃高光层（左上角白色椭圆高光）
3. 内部星光粒子（小型六芒星/四芒星闪光点，约 3～5 个）
4. 边缘光晕描边（白色细边）
5. 外部发光阴影（紫粉色柔和投影）

**气泡尾部**：
- 位于心形底部右侧，类似对话气泡的「尾巴」
- 强化「聊天/对话」的产品属性暗示
- 形状：向右下方微微延伸的弧形尖角

**状态**：
| 状态 | 表现 |
|------|------|
| 默认 | 静态（设计稿状态）|
| 进入动画 | 弹性放大（scale 0.8 → 1.0，spring 缓动）|
| 持续动画（推荐）| 轻微上下浮动（±8px，3s 周期）|

**复用建议**：**强烈建议复用**。作为 `<AppGemIcon>` 组件（或 `<img src="gem-icon.png">`），用于：
- 启动屏（当前）
- App 图标
- 个人页 / 设置页品牌区域
- 空状态插图配合使用

### 5. AppWordmark（应用字标）

**作用**：展示 App 名称 "yuoyuo"（全小写）。

| 属性 | 值 |
|------|-----|
| 层级（z-index） | 10 |
| 内容 | `yuoyuo`（7个字母，全小写）|
| 字体 | SF Pro Rounded / 圆体变体 |
| 字号（画布px） | ≈ 96px（估算值）|
| 字号（逻辑pt） | ≈ 37pt（估算值）|
| 字重 | Bold（700）|
| 颜色 | `#3A3A4A`（深炭色）|
| 行高 | 1.0（紧凑）|
| 字距 | ≈ -0.02em（轻微收紧，估算值）|
| 对齐 | 水平居中 |
| 背景 | 透明（直接在渐变背景上）|
| 装饰 | 无下划线、无阴影 |

**视觉特征**：
- 字体选择「SF Pro Rounded」，字母末端均为圆弧收笔，与产品的「软糯治愈」调性高度一致
- 深炭色 `#3A3A4A` 与粉色背景对比清晰，但不显刺眼（非纯黑）
- 字标无额外装饰（无花体、无渐变填充），保持简洁

**状态**：
| 状态 | 表现 |
|------|------|
| 默认 | 静态显示 |
| 进入动画（推荐）| Fade in + Slide up（从 Y+16px 滑入）|

**复用建议**：**强烈建议复用**。作为 `<AppWordmark>` 组件，用于 App 顶栏、关于页、品牌展示等场景。

---

### 6. AppTagline（应用 Tagline）

**作用**：品牌 Tagline「陪你聊聊吧」，传达产品价值主张。

| 属性 | 值 |
|------|-----|
| 层级（z-index） | 10 |
| 内容 | `陪你聊聊吧`（5个汉字）|
| 字体 | PingFang SC（Regular）|
| 字号（画布px） | ≈ 48px（估算值）|
| 字号（逻辑pt） | ≈ 18pt（估算值）|
| 字重 | Regular（400）|
| 颜色 | `#5A5A6A`（中灰炭色，比字标略浅，估算值）|
| 行高 | 1.5 |
| 字距 | ≈ 0.12em（宽松，字间有明显空隙，估算值）|
| 对齐 | 水平居中 |

**视觉特征**：
- 字距明显比普通正文宽，产生「舒展、从容」的视觉感
- 颜色比字标浅，形成视觉层级（字标 > tagline）
- 中文选用 PingFang SC，与品牌的温和气质一致

**状态**：
| 状态 | 表现 |
|------|------|
| 默认 | 静态显示 |
| 进入动画（推荐）| Fade in（晚于字标 100ms 出现）|

**复用建议**：中等复用性。可提取为 `<AppTagline>` 组件，但 tagline 文案可能随版本迭代调整。

---

### 7. BreathingDots（底部呼吸点指示器）

**作用**：底部三个圆点组成的加载指示器，通过「呼吸动画」暗示系统后台正在初始化，降低用户等待焦虑。

| 属性 | 值 |
|------|-----|
| 层级（z-index） | 20（最顶层）|
| 数量 | 3 个圆点（左、中、右）|
| 排列 | 水平排列，水平居中于画面 |
| 间距（gap） | ≈ 12px（估算值）|

**各圆点规格**：

| 圆点 | 直径（画布px）| 颜色 | 透明度 | 说明 |
|------|---------------|------|--------|------|
| 左侧点 | ≈ 12px（估算值）| `#F0A0B8`（估算值）| 0.6 | 较小，较暗 |
| 中间点 | ≈ 18px（估算值）| `#E87A9A`（估算值）| 1.0 | 最大，最亮，当前激活 |
| 右侧点 | ≈ 12px（估算值）| `#F0A0B8`（估算值）| 0.6 | 较小，较暗 |

> 注意：从 PNG 观察，中间点明显比两侧点大且颜色更深/更鲜艳，暗示这是一个「分页指示器」或「加载进度点」，中间为当前状态。

**动画规格（设计稿未定义，参考推荐）**：
- 三点依次执行 scale（0.7→1.2→0.7）+ opacity（0.5→1.0→0.5）的循环动画
- 相邻点之间延迟 200ms，产生波浪感
- 动画周期：1200ms / 每点
- Easing：`ease-in-out`

**状态**：
| 状态 | 表现 |
|------|------|
| 加载中 | 三点持续呼吸动画（循环）|
| 加载完成 | Fade out（随启动屏整体消失）|

**复用建议**：**建议复用**。作为 `<BreathingDots>` 或 `<LoadingDots>` 组件，可用于：
- 启动屏（当前）
- 消息列表加载中状态
- AI 回复等待中状态（对话界面）

## 组件复用优先级汇总

| 优先级 | 组件 | 理由 |
|--------|------|------|
| P0（必须做成组件）| `GemIcon` | 品牌核心图标，多处复用 |
| P0（必须做成组件）| `AppWordmark` | 品牌名称，全局使用 |
| P1（建议做成组件）| `BreathingDots` | 加载状态通用需求 |
| P2（可选组件）| `AppTagline` | 文案可能变化，但结构可复用 |
| P3（内联实现）| `SplashBackground` | 场景专属，不通用 |
| P3（内联实现）| `CloudDecoration` | 场景专属装饰 |
| P3（内联实现）| `TopGlowEffect` | 场景专属装饰 |
