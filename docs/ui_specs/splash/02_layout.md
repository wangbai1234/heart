# 02 布局规范 — Splash Screen（启动屏）

## 画布规格

| 属性 | 值 |
|------|-----|
| 画布尺寸（设计稿） | 1024 × 1536 px |
| 逻辑尺寸（iPhone 14） | 390 × 844 pt |
| 像素比例（DPR） | ~2.625×（1024/390 ≈ 2.626） |
| 安全区顶部 | 47 pt（逻辑）≈ 123 px（画布） |
| 安全区底部 | 34 pt（逻辑）≈ 89 px（画布） |
| 方向 | 仅竖屏（Portrait） |

---

## 整体层级结构

启动屏采用**全屏叠层（Full-screen Stack）**布局，从底到顶共5个层级：

```
z-index 层级（从低到高）
┌─────────────────────────────────────┐
│  z:0  背景渐变层（Background Gradient）│
├─────────────────────────────────────┤
│  z:1  云朵装饰层（Cloud Decorations） │
├─────────────────────────────────────┤
│  z:2  顶部光晕层（Top Glow Effect）   │
├─────────────────────────────────────┤
│  z:3  内容层（Content Layer）         │
│       ├── 心形宝石图标                │
│       ├── yuoyuo 字标                 │
│       └── 陪你聊聊吧 tagline          │
├─────────────────────────────────────┤
│  z:4  底部指示器层（Bottom Dots）     │
└─────────────────────────────────────┘
```

## 区域划分（画布坐标，1024×1536 px）

### 1. 全屏背景区域（Background Zone）

| 属性 | 值 |
|------|-----|
| 位置 | (0, 0) |
| 尺寸 | 1024 × 1536 px |
| 内容 | 渐变天空 + 云朵装饰 |

### 2. 安全区域（Safe Area）

| 属性 | 值 |
|------|-----|
| 顶部安全区高度 | 123 px（估算值） |
| 底部安全区高度 | 89 px（估算值） |
| 可用内容区域 | 1024 × (1536 - 123 - 89) = 1024 × 1324 px |

### 3. 主内容区（Content Zone）

内容区垂直居中于画布，核心元素群（图标 + 字标 + tagline）整体：

| 属性 | 值 |
|------|-----|
| 水平对齐 | 画布水平居中（center） |
| 垂直位置 | 约 Y 400～950 px（估算值，视觉重心略偏上） |
| 宽度约束 | 最大宽度 ≈ 700 px（估算值），两侧留白 ≈ 162 px |

## Flex / Grid 布局描述

启动屏整体采用 **Flex Column + Center Alignment** 的单列垂直布局：

```
Page (position: fixed, width: 100vw, height: 100vh)
  └── Background Layer (position: absolute, inset: 0)
  └── Content Stack (
        display: flex,
        flex-direction: column,
        align-items: center,
        justify-content: center,   /* 整体垂直居中，视觉重心略偏上 */
        gap: 自定义间距
      )
        ├── GemIcon
        ├── Wordmark
        └── Tagline
  └── BottomIndicator (
        position: absolute,
        bottom: safe-area-bottom + offset,
        left: 50%,
        transform: translateX(-50%)
      )
```

## 滚动行为

启动屏为**全屏静态/动画展示页**，无滚动。禁用所有滚动交互。
