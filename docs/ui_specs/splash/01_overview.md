# 01 概览 — Splash Screen（启动屏）

## 页面基本信息

| 字段 | 值 |
|------|-----|
| 页面名称 | Splash Screen / 启动屏 |
| 页面 ID | splash |
| Phase | Phase 1 — Foundation |
| 画布尺寸 | 1024 × 1536 px（逻辑尺寸 390 × 844 pt，iPhone 14） |
| 设计工具 | Figma（推测）/ 对应 Capacitor 启动屏 |
| 最后更新 | 2026-06-28 |

---

## 页面作用

Splash Screen 是用户每次冷启动 yuoyuo App 时最先看到的全屏视觉界面。其核心作用为：

1. **品牌第一印象**：在应用初始化（JS bundle 加载 / DB 连接）的 1～3 秒内呈现品牌视觉，避免白屏体验。
2. **情绪锚定**：通过梦幻渐变天空 + 宝石心形图标，在用户进入对话之前建立「治愈、温暖、陪伴」的情绪基调。
3. **品牌字标展示**：确立 yuoyuo 品牌名称（全小写）+ tagline「陪你聊聊吧」的视觉规范。
4. **加载进度指示**：底部三点呼吸动画（breathing dots）向用户暗示系统正在加载，降低等待焦虑。

---

## 目标

- **品牌记忆**：用户能在 3 秒内记住 yuoyuo 的核心视觉符号（心形宝石 gem icon）
- **情绪传递**：让用户在等待期间感受到「被陪伴」的温暖，而非焦虑等待
- **无感过渡**：从启动屏到主界面的转场尽量丝滑，不产生视觉断层

---

## 进入方式

| 触发条件 | 说明 |
|----------|------|
| 冷启动（Cold Launch） | 用户从系统桌面点击 App 图标，操作系统加载进程，第一帧即显示此屏 |
| Capacitor 启动屏配置 | 由 `@capacitor/splash-screen` 插件控制，使用本图作为 `launchShowDuration` 期间的静态/动态资源 |
| 强制杀进程后重启 | 同冷启动 |

不适用场景：后台切换回 App（warm resume）通常跳过启动屏。

---

## 退出方式

| 退出触发 | 过渡效果 |
|----------|----------|
| 应用初始化完成（JS ready）| Fade out → 主界面 Fade in（建议 300ms ease-out） |
| 超时兜底（≥3s）| 强制 Fade out，防止卡死在启动屏 |
| 自动消失 | 不依赖用户操作，系统自动管理 |

---

## 用户流程

```
系统桌面
    ↓ 点击 App 图标
Splash Screen 显示（0ms）
    ↓ 底部呼吸点动画循环
JS Bundle / DB 初始化中...
    ↓ 初始化完成 or 超时
Fade Out（300ms）
    ↓
主界面 / 登录页（取决于用户登录状态）
```

## 设计理念

### 核心哲学：「云端相遇的第一眼」

启动屏将用户带入一片粉紫渐变的梦幻天空，仿佛正从云端降落，即将与 AI 伴侣 yuoyuo 相遇。这一视觉叙事与产品核心价值「AI 陪伴」高度契合——不是冰冷的工具启动，而是一次温柔的情感邀请。

### 设计语言三层叠加

1. **背景层**：扩散式渐变天空（粉→奶油→粉紫），大面积柔和色块营造无压力的视觉空间。云朵装饰（低饱和度、高模糊）增加层次感，避免背景过于单调。
2. **图标层**：中央玻璃质感心形宝石（左粉右紫蓝，高光 + 内部星光粒子），以3D拟物风格传递「珍贵、有温度、有灵魂」。心形底部带有气泡对话框的尾巴，暗示「聊天陪伴」属性。
3. **文字层**：品牌字标 "yuoyuo"（深炭色，粗圆体）+ tagline「陪你聊聊吧」（细体，字距宽松），简洁有力，避免信息过载。

### 底部呼吸点

三个小圆点（粉红色，中间一个更大/更亮），以「呼吸节律」动画暗示系统活跃，将「等待」转化为「期待」的心理体验。

## 视觉关键词

| 关键词 | 表现形式 |
|--------|----------|
| **Glassmorphism** | 心形宝石的玻璃透明质感、高光反射、折射内部光晕 |
| **Anime / 二次元** | 渐变天空配色、云朵风格、整体插画质感 |
| **Warm（温暖）** | 粉色主调、奶油色高光、整体偏暖的色温 |
| **Healing（治愈）** | 大面积留白、柔和渐变、无锐角设计 |
| **Dreamy（梦幻）** | 天空背景的云雾感、粉紫渐变、星光粒子 |
| **Soft（柔软）** | 低对比度文字、圆润字体、云朵模糊边缘 |
| **Minimal（极简）** | 信息元素仅三层（图标/字标/tagline），无多余装饰 |

## 情绪传达

启动屏在 1～3 秒内完成以下情绪旅程：

```
[初见] 温柔的粉色包裹感 → [发现] 闪亮的宝石心形让眼睛一亮
→ [认同] "yuoyuo" 字样带来品牌记忆点
→ [期待] "陪你聊聊吧" 触发「被陪伴」的心理需求
→ [信任] 底部呼吸点传达系统正在认真准备中
```

目标情绪：**安心、期待、被接纳、温柔兴奋**。
