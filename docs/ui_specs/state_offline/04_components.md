# 04 Components — 离线降级状态页

---

## C-01 状态栏 (Status Bar)

| 属性 | 值 |
|------|----|
| 名称 | StatusBar |
| 作用 | 显示系统时间、信号、电池状态 |
| 层级 | Z-index 最高，固定顶部 |
| 尺寸 | W: 390 pt，H: 47 pt |
| 背景 | 透明，继承页面渐变 |
| 内容 | 左: 「9:41」时间；右: 信号条（深色） + WiFi（深色） + 电池（满格，深色） |
| 状态 | 系统控制，非自定义 |
| 可复用 | 全应用共用，勿自行实现 |

---

## C-02 导航栏 (Navigation Bar)

| 属性 | 值 |
|------|----|
| 名称 | ChatNavBar |
| 作用 | 展示对话对象信息 + 操作入口；**离线时额外展示副标题状态文字** |
| 层级 | 固定顶部，StatusBar 下方 |
| 尺寸 | W: 390 pt，H: ≈ 56 pt（估算值） |
| 背景 | 透明，无边框线 |
| Padding | H: 16 pt，V: 8 pt |

### 子组件

**C-02-A 返回按钮**
- 图标：`<` 箭头，24×24 pt，颜色 #3A3A4A
- 触摸目标：44×44 pt（含 padding 扩大）
- 操作：点击返回上一页

**C-02-B 头像**
- 形状：圆形 clip，40×40 pt
- 图片：AI 角色 悠悠 头像（动漫风，紫/棕发，梦幻感）
- 边框：无
- 紧邻返回按钮右侧，间距 8 pt

**C-02-C 文字区**
- 布局：垂直堆叠，居左
- 主标题「悠悠」：17 pt SemiBold，#3A3A4A
- 副标题「离线 · 消息将自动发送」：12 pt Regular，#A0A0B0（估算值），主标题下 2 pt
- 副标题颜色在离线状态下变为浅灰，**在线状态下为空或显示「在线」绿色**

**C-02-D 电话图标按钮**
- 图标：电话话筒，24×24 pt，颜色 #3A3A4A
- 触摸目标：44×44 pt
- 距右边 16 pt，垂直居中

**C-02-E 更多菜单按钮**
- 图标：`···`（三点），24×24 pt，颜色 #3A3A4A
- 触摸目标：44×44 pt
- 位于电话图标左侧，间距 16 pt（估算值）

### 状态变体
| 状态 | 副标题文字 | 副标题颜色 |
|------|-----------|-----------|
| 在线（正常） | 空/「在线」 | `#66BB6A`（绿色） |
| **离线（当前）** | **「离线 · 消息将自动发送」** | **`#A0A0B0`（灰）** |
| 重连中 | 「重新连接中…」 | `#FFA726`（橙色）|

### 可复用
是，整个应用聊天页共用，通过 Props 切换副标题文字与颜色。

## C-03 离线横幅 (Offline Banner)

| 属性 | 值 |
|------|----|
| 名称 | OfflineBanner |
| 作用 | 非强制性弱网提示，同时提供手动重试入口 |
| 层级 | 历史消息区上方，横向 Margin 16 pt 卡片 |
| 尺寸 | W: ≈ 358 pt（估算值），H: ≈ 52 pt（估算值） |
| Margin Top | ≈ 8 pt（导航栏下） |
| Padding | H: 16 pt，V: 12 pt |
| Border Radius | 16 pt |
| 背景 | `rgba(255,255,255,0.85)` + `backdrop-filter: blur(8pt)` |
| 阴影 | `0 2pt 8pt rgba(0,0,0,0.06)` |
| 布局 | 横排，三元素：图标 + 文字（flex:1）+ 重试按钮；垂直居中 |

### 子组件

**C-03-A 云朵图标**
- 类型：emoji 或自绘 SVG 云朵
- 尺寸：约 24×20 pt（估算值）
- 颜色：粉色系 `#FFB7C5`
- 含义：柔和的网络不稳定隐喻（非错误感）

**C-03-B 说明文字**
- 文字：「网络不太稳定，刚才的消息已暂存。」
- 字号：14 pt Regular，#3A3A4A
- 行数：单行（在正常宽度下不换行）
- flex: 1，撑满图标与重试按钮之间

**C-03-C 重试按钮 (RetryButton)**
- 文字：「↻ 重试」（刷新图标 + 文字）
- 宽度：≈ 68 pt（估算值），高度：≈ 32 pt（估算值）
- Border Radius：16 pt（pill 型）
- 背景：透明
- 边框：1 pt，颜色 #FFB7C5
- 文字颜色：#FFB7C5，13 pt Medium
- Hover/Pressed：背景变为 `rgba(255,183,197,0.15)`，scale 0.96
- 触摸目标：≥ 44×44 pt（需 padding 扩大）

### 状态
| 状态 | 样式 |
|------|------|
| 默认 | 如上规格 |
| 重试中（loading） | 「↻ 重试」图标旋转动画，按钮禁用，opacity 0.6 |
| 重连成功 | 横幅整体淡出消失（opacity 0→1 reverse，上滑 translateY -100%） |

### 可复用
是，可提取为通用 `NetworkBanner` 组件，接收 `message`、`onRetry`、`isRetrying` Props。

## C-04 历史消息列表 (Chat History List)

| 属性 | 值 |
|------|----|
| 名称 | ChatHistoryList |
| 作用 | 展示历史对话记录；**离线时整体 opacity 降低为 0.92** |
| 层级 | 横幅下方，Composer 上方，可滚动 |
| 宽度 | 390 pt（全宽） |
| Padding H | 16 pt |
| 整体 Opacity（离线） | 0.92 |
| 滚动 | 垂直滚动，iOS rubber-band |

### 子组件

**C-04-A 时间戳标签 (Timestamp Label)**
- 文字格式：「昨天 22:18」等
- 字号：12 pt，颜色 #ACACAC
- 对齐：水平居中
- Margin：上 12 pt，下 8 pt（估算值）

**C-04-B AI 消息气泡 (AI Bubble)**
- 布局：左对齐，头像 + 气泡横排
- 头像：圆形 36×36 pt，悠悠 AI 头像图，左侧对齐
- 气泡：
  - 背景：`rgba(255,255,255,0.90)`
  - Border Radius：TL 4 pt / 其余 16 pt
  - Padding：H 12 pt，V 10 pt
  - 最大宽度：250 pt（估算值）
  - 文字：15 pt Regular，#3A3A4A，行高 22 pt
  - 含 emoji：内联渲染

**C-04-C 用户消息气泡 (User Bubble)**
- 布局：右对齐，无头像
- 气泡：
  - 背景：`linear-gradient(135deg, #FFE4EC → #FFDDE8)`（估算值）
  - Border Radius：TL 16 pt / TR 4 pt / BR 16 pt / BL 16 pt
  - Padding：H 12 pt，V 10 pt
  - 最大宽度：250 pt（估算值）
  - 文字：15 pt Regular，#3A3A4A，行高 22 pt
- 双勾状态图标（✓✓）：字号 14 pt，颜色 #A7C7E7，气泡右下角外 4 pt

**C-04-D 心形 emoji 装饰**
- 在 AI 气泡右下角外（消息「今天过得怎么样呀？有没有遇到什么开心的事～」下方）
- 粉红色心形 ♥，约 16 pt
- 略微旋转约 -10°（估算值），增加手绘感

### 状态变体
| 状态 | Opacity |
|------|---------|
| 正常在线 | 1.0 |
| **离线（当前）** | **0.92** |
| 网络极差（更深暗淡，未在此稿体现） | 0.80（设计扩展建议） |

## C-05 暂存消息分割线 (Staged Messages Divider)

| 属性 | 值 |
|------|----|
| 名称 | StagedDivider |
| 作用 | 视觉上分隔历史已发送消息与本地暂存消息；说明暂存逻辑 |
| 层级 | 历史消息列表底部，消息流内嵌 |
| 宽度 | 全宽 390 pt（文字居中） |
| Height | ≈ 56 pt（估算值） |
| Padding H | 32 pt（估算值） |
| Padding V | 8 pt |

### 子元素
| 元素 | 规格 |
|------|------|
| 第一行「以下为暂存消息」 | 13 pt Regular，#C0A0A0（估算值），居中 |
| 第二行「网络恢复后会自动发送」 | 12 pt Regular，#C0A0A0（估算值），居中，上方间距 4 pt |
| 分隔线 | 无水平线，仅文字视觉分隔 |

### 状态
- **无暂存消息**：显示（作为提示，下方暂存区为空）
- **有暂存消息**：显示，暂存消息列于其下

### 可复用
是，可复用于「发送中」「已失败」等其他消息状态分区场景。

## C-06 Composer 输入栏 (Composer)

| 属性 | 值 |
|------|----|
| 名称 | Composer |
| 作用 | 用户输入新消息；**离线时右侧显示时钟角标，表示消息将暂存** |
| 层级 | 固定底部，Safe Area Bottom 上方 |
| 宽度 | 390 pt |
| 高度 | ≈ 56 pt（估算值，不含 Safe Area） |
| 背景 | `rgba(255,255,255,0.80)` + `backdrop-filter: blur(8pt)` |
| 顶部分隔线 | 0.5 pt，`rgba(0,0,0,0.08)` |
| Padding H | 16 pt |
| Padding V | 8 pt |
| 布局 | 横排：[+ 按钮] [输入框 flex:1] [时钟/发送按钮] |
| 元素间距 | 8 pt（估算值） |

### 子组件

**C-06-A 添加按钮 (AddButton)**
- 形状：圆形，约 36×36 pt
- 图标：`+` 号，约 20×20 pt，颜色 #A0A0A8
- 边框：1 pt，颜色 #D0D0D0（估算值）
- 背景：透明/白色半透明
- 触摸目标：44×44 pt

**C-06-B 输入框 (TextInput)**
- 宽度：flex:1（撑满中间区域）
- 高度：≈ 36 pt（估算值）
- Border Radius：18 pt（pill）
- 背景：`rgba(255,255,255,0.60)`
- Placeholder：「想对悠悠说点什么呢...」，15 pt Regular，#C0C0C8
- 正文：15 pt Regular，#3A3A4A
- Padding H：12 pt（估算值）

**C-06-C 时钟角标按钮 (ClockBadgeButton)**

> **核心离线状态指示组件**

- 形状：圆形，约 36×36 pt
- 背景：`#FFB7C5`（粉色实色填充）
- 图标：时钟/钟表轮廓，约 20×20 pt，颜色 #FFFFFF（白色）
- 阴影：`0 2pt 6pt rgba(255,183,197,0.40)`（粉色发光投影，估算值）
- 触摸目标：44×44 pt
- 作用：表示当前输入将以暂存形式保存；点击可查看暂存列表（设计稿未定义交互，待确认）
- **在线状态下**：替换为标准发送按钮（箭头/纸飞机图标）

### 状态变体
| 场景 | 右侧按钮 |
|------|---------|
| 在线，输入框空 | 发送按钮（disabled 状态，灰色）|
| 在线，有输入内容 | 发送按钮（active 状态，粉色填充） |
| **离线（当前）** | **时钟角标按钮（粉色填充，白色时钟图标）** |

### 可复用
是，Composer 为全局组件，通过 `mode` Prop（`online` / `offline`）切换右侧按钮形态。
