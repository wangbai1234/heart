# 06 Interactions — 离线降级状态页

---

## 触发条件（进入离线状态）

| 事件 | 条件 |
|------|------|
| WebSocket 重连失败 | 累计 ≥ 3 次失败 |
| 触发时机 | 第 3 次失败确认后，无延迟立即触发 |
| 静默期 | 前 2 次重连失败时不显示任何 UI 变化（静默重试） |

---

## 进入动画序列（进入离线状态时）

### Step 1 — 历史消息区暗淡
| 属性 | 值 |
|------|-----|
| 目标 | 历史消息区整体 Opacity |
| 变化 | `1.0` → `0.92` |
| 时长 | 400 ms |
| 曲线 | `ease-out` |
| 延迟 | 0 ms |

### Step 2 — 横幅下滑入场
| 属性 | 值 |
|------|-----|
| 目标 | OfflineBanner |
| 初始状态 | `opacity: 0; translateY: -20pt` |
| 结束状态 | `opacity: 1; translateY: 0` |
| 时长 | 300 ms |
| 曲线 | `cubic-bezier(0.34, 1.56, 0.64, 1.0)`（带轻微回弹） |
| 延迟 | 0 ms（与历史区暗淡同时） |

### Step 3 — 导航栏副标题切换
| 属性 | 值 |
|------|-----|
| 目标 | ChatNavBar 副标题文字 |
| 动画 | 淡出旧文字 → 淡入「离线 · 消息将自动发送」 |
| 时长 | 各 150 ms（out + in = 300 ms 总计） |
| 曲线 | `ease-in` 淡出，`ease-out` 淡入 |
| 延迟 | 0 ms |

### Step 4 — 时钟角标按钮弹入
| 属性 | 值 |
|------|-----|
| 目标 | Composer 右侧按钮（发送 → 时钟切换） |
| 初始状态 | `scale: 0.5; opacity: 0` |
| 结束状态 | `scale: 1.0; opacity: 1` |
| 时长 | 200 ms |
| 曲线 | `cubic-bezier(0.34, 1.56, 0.64, 1.0)`（弹性放大） |
| 延迟 | 100 ms（历史区暗淡后触发，避免同时太多动画） |

### Step 5 — 暂存分割线淡入
| 属性 | 值 |
|------|-----|
| 目标 | StagedDivider 区域 |
| 初始状态 | `opacity: 0` |
| 结束状态 | `opacity: 1` |
| 时长 | 250 ms |
| 曲线 | `ease-out` |
| 延迟 | 150 ms |

---

## 退出动画序列（恢复在线时）

### 顺序（与进入动画相反）
1. 横幅上滑淡出（translateY → -20pt，opacity → 0，200 ms ease-in）
2. 时钟角标按钮缩小淡出（scale → 0.5，opacity → 0，150 ms）→ 发送按钮弹入
3. 导航栏副标题切换回在线文字（同进入动画文字淡切逻辑）
4. 历史消息区 opacity 回升至 1.0（400 ms ease-out）
5. 暂存分割线淡出（200 ms ease-in，同时暂存消息状态变为「发送中」）

## 各组件交互规格

### OfflineBanner（整体）
| 状态 | 表现 |
|------|------|
| 默认 | 静态显示 |
| 无交互 | 横幅本身不可点击（仅内部按钮可点击） |

### 重试按钮 (RetryButton)
| 事件 | 表现 |
|------|------|
| Hover（移动端不适用，仅文档备注） | 背景变为 `rgba(255,183,197,0.15)` |
| Pressed | scale 缩放至 0.96，duration 100 ms，ease-in |
| Released | scale 回 1.0，duration 150 ms，spring |
| 点击后（重试中） | 按钮 disabled，刷新图标开始旋转 `spin 1s linear infinite`，opacity 降至 0.6 |
| 重连成功 | 整体退出离线状态 |
| 重连再次失败 | 停止旋转，按钮恢复可点击状态，可选：横幅文字短暂更新为「重连失败，将自动重试」 |

### 时钟角标按钮 (ClockBadgeButton)
| 事件 | 表现 |
|------|------|
| Pressed | scale 缩放至 0.94，duration 80 ms |
| Released | scale 回 1.0，duration 120 ms，spring |
| 点击后 | 设计稿未定义，建议：展示暂存消息预览 Sheet 或 Toast 提示「已暂存 X 条消息」 |

### 添加按钮 (+)
| 事件 | 表现 |
|------|------|
| Pressed | scale 0.96，duration 100 ms |
| 点击后 | 展开媒体选择 Sheet（行为同在线状态，但上传在离线时会被暂存） |
| 备注 | 离线状态下附件同样可以暂存（设计稿未明确，待确认） |

### 输入框 (TextInput)
| 事件 | 表现 |
|------|------|
| 聚焦 | 软键盘弹起，Composer 和 消息列表 整体上移 |
| 输入文字 | 正常输入，无特殊限制 |
| 发送（按回车/点击时钟按钮） | 消息以暂存状态追加至分割线以下的消息列表，显示「已暂存」状态标记 |

### 历史消息列表
| 事件 | 表现 |
|------|------|
| 滚动 | 正常滚动，opacity 0.92 不影响交互 |
| 长按消息气泡 | 展示消息操作菜单（复制/删除等，同在线状态） |

## 持续状态动效（离线期间一直运行）

### 无持续动效（设计稿未见）
当前设计稿在离线稳定状态下没有持续循环动效（无呼吸光效、无骨架 shimmer）。

### 建议扩展（设计稿未定义，开发可选实现）
| 动效 | 描述 | 触发条件 |
|------|------|---------|
| 时钟角标呼吸光效 | 时钟按钮阴影在 `rgba(255,183,197,0.3)` 至 `rgba(255,183,197,0.6)` 之间循环，4 s loop | 离线状态持续 > 5 s |
| 云朵图标微晃 | 水平方向 ±2 pt 缓慢摆动，8 s loop | 离线状态持续 > 10 s |

## 状态机图

```
[在线聊天状态]
    │
    │ WS 重连失败 × 1, 2（静默，无 UI）
    │
    │ WS 重连失败 × 3
    ▼
[离线降级状态] ◀─────────────────────┐
    │                               │
    ├─ 用户点击「重试」               │
    │      └─ 重连中状态（按钮 loading）│
    │            ├─ 成功 → [在线聊天状态]
    │            └─ 失败 → [离线降级状态]（循环）
    │                               │
    ├─ 用户继续输入                   │
    │      └─ 消息暂存到本地           │
    │                               │
    ├─ 网络自动恢复                   │
    │      └─ WS 重连成功 → 消息批量发出 → [在线聊天状态]
    │                               │
    └─ 用户退出页面                   │
           └─ 暂存消息保留，下次进入自动补发
```

## 设计稿未定义的交互（需产品确认）

| 问题 | 建议处理 |
|------|---------|
| 时钟角标点击后的跳转/弹层行为 | 建议：Toast 「已暂存 X 条消息，网络恢复后自动发送」 |
| 离线状态下能否拍照/录音 | 建议：允许，但提示「将在网络恢复后发送」 |
| 多条暂存消息如何在列表中展示 | 建议：分割线下方，按输入顺序排列，状态标记「已暂存」 |
| 离线持续超过 N 分钟的降级处理 | 建议：出现「离线时间较长」二级提示 |
| 暂存消息发送失败（服务器拒绝）后如何处理 | 建议：单条消息级别的错误提示，独立重试按钮 |
