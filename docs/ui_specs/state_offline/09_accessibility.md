# 09 Accessibility — 离线降级状态页

---

## 触摸目标尺寸（Touch Targets）

iOS HIG 最低要求：44×44 pt

| 元素 | 视觉尺寸 | 触摸目标 | 状态 |
|------|---------|---------|------|
| 返回按钮 `<` | 24×24 pt | 44×44 pt（含 padding） | ✅ 合规 |
| 头像（导航栏，若可点击） | 40×40 pt | 44×44 pt（含 padding 2pt） | ✅ 合规 |
| 电话图标按钮 | 22×22 pt | 44×44 pt（含 padding） | ✅ 合规 |
| 更多菜单按钮 `···` | 22×22 pt | 44×44 pt（含 padding） | ✅ 合规 |
| 重试按钮 | 68×32 pt（视觉） | 建议扩展至 ≥44 pt 高度（含 padding V:6pt） | ⚠️ 需确认 |
| 时钟角标按钮 | 36×36 pt | 44×44 pt（需 padding 4pt） | ⚠️ 需确认 |
| 添加按钮 `+` | 36×36 pt | 44×44 pt（需 padding 4pt） | ⚠️ 需确认 |
| 输入框 | flex 填满，H≈36pt | 44 pt（需 padding 扩展） | ⚠️ 需确认 |
| 消息气泡（长按） | 可变宽度，H≥40pt | 满足（自然高度） | ✅ 合规 |

> 带 ⚠️ 的元素在视觉上小于 44 pt，实现时须通过 hitSlop 或 padding 扩大触摸区域。

## 颜色对比度（WCAG 2.1）

### 文字对比度要求
- 正常文字（< 18pt）：最低 4.5:1
- 大文字（≥ 18pt 或 ≥ 14pt Bold）：最低 3.0:1

| 元素 | 前景色 | 背景色 | 估算对比度 | WCAG AA |
|------|--------|--------|-----------|---------|
| 消息正文 `#3A3A4A` / 白气泡 | #3A3A4A | #FFFFFF | ≈ 9.8:1 | ✅ |
| 消息正文 `#3A3A4A` / 粉气泡 `#FFE4EC` | #3A3A4A | #FFE4EC | ≈ 8.2:1 | ✅ |
| Nav 主标题 `#3A3A4A` / 透明背景（奶油白） | #3A3A4A | #FFF8F3 | ≈ 9.5:1 | ✅ |
| Nav 副标题 `#A0A0B0` / 奶油白 | #A0A0B0 | #FFF8F3 | ≈ 3.2:1 | ⚠️ 接近临界（AA Normal 不合规，AA Large 合规） |
| 横幅文字 `#3A3A4A` / 白色横幅 | #3A3A4A | #FFFFFF | ≈ 9.8:1 | ✅ |
| 重试按钮文字 `#FFB7C5` / 透明背景（奶油白） | #FFB7C5 | #FFF8F3 | ≈ 1.8:1 | ❌ 不合规（仅装饰性按钮边界，建议加深颜色至 #E8799A） |
| 时间戳 `#ACACAC` / 奶油白 | #ACACAC | #FFF8F3 | ≈ 2.8:1 | ⚠️ 不合规（时间戳属次要信息，设计上接受，建议加深至 #868686） |
| 暂存分割线 `#C0A0A0` / 奶油白 | #C0A0A0 | #FFF8F3 | ≈ 3.0:1 | ⚠️ AA Large 合规 |
| Placeholder `#C0C0C8` / 输入框 | #C0C0C8 | #FFFFFF | ≈ 2.1:1 | ❌ Placeholder 豁免（WCAG 1.4.3 Exception for placeholders） |

> **优先修复项**：重试按钮文字颜色对比度不足（1.8:1），若按钮文字为可操作内容，应加深至 WCAG AA 合规。

## ARIA 语义（Screen Reader 支持）

### 页面级
```
role="main" — 整个聊天内容区域
aria-label="与悠悠的对话"
```

### 离线状态横幅 OfflineBanner
```
role="status"                              — 非紧急状态通知（不打断 SR 朗读）
aria-live="polite"                         — 状态变化时礼貌通知
aria-label="网络离线。网络不太稳定，刚才的消息已暂存。"
```

> 选择 `role="status"` 而非 `role="alert"`：避免打断用户正在进行的输入操作。弱网非紧急情况，不应强制 Screen Reader 立即播报。

### 重试按钮
```
role="button"
aria-label="重试连接"
aria-busy="false"                          — 重连中时改为 aria-busy="true"
```

### 导航栏副标题（离线状态）
```
aria-label="悠悠，离线状态，消息将自动发送"   — 将主标题和副标题合并为一个可读描述
```

### 时钟角标按钮（Composer）
```
role="button"
aria-label="消息暂存中，点击查看暂存消息"
aria-pressed="false"
```

### 历史消息列表
```
role="log"                                 — 聊天记录使用 log 角色
aria-label="对话历史记录"
aria-live="off"                            — 历史消息不需要实时播报
```

### 单条消息气泡
```
<!-- AI 消息 -->
role="article"
aria-label="悠悠 昨天22:18 说：今天过得怎么样呀？有没有遇到什么开心的事～"

<!-- 用户消息 -->
role="article"
aria-label="你 昨天22:21 说：还不错！下班路上看到超美的晚霞，感觉一天的疲惫都被治愈了"
```

### 已读双勾
```
aria-label="已读"
aria-hidden="false"                        — 需对 SR 可见，表示消息已读状态
```

### 暂存分割线
```
role="separator"
aria-label="以下为暂存消息，网络恢复后会自动发送"
```

### 输入框
```
role="textbox"
aria-label="输入消息"
aria-multiline="false"
placeholder="想对悠悠说点什么呢..."
```

### 添加按钮
```
role="button"
aria-label="添加附件"
```

## Screen Reader 状态播报策略（重点）

### 进入离线状态时的 SR 播报顺序
1. **延迟 300 ms**（等进入动画完成）
2. 通过 `aria-live="polite"` 的 OfflineBanner 礼貌播报：
   「网络不太稳定，刚才的消息已暂存。」
3. 用户聚焦输入框时，Composer 的 `aria-label` 更新为：
   「输入消息，当前处于离线状态，消息将暂存后自动发送」
4. **不播报**历史消息区暗淡变化（视觉降级，无语义变化）

### 重连状态变化时的 SR 播报
| 状态变化 | 播报内容 | 播报方式 |
|---------|---------|---------|
| 点击重试后 | 「正在重新连接...」 | `aria-live="polite"` |
| 重连成功 | 「已连接，暂存消息正在发送」 | `aria-live="polite"` |
| 重连再次失败 | 「重连失败，将继续自动重试」 | `aria-live="polite"` |

## 键盘导航（iPad / 外接键盘场景）

| 操作 | 行为 |
|------|------|
| Tab | 依次聚焦：返回按钮 → 头像 → 电话 → 更多 → 重试按钮 → 输入框 → 添加按钮 → 时钟按钮 |
| Shift+Tab | 反向聚焦 |
| Enter / Space | 激活当前聚焦按钮 |
| Escape | 关闭软键盘（输入框失焦）/ 返回（若无软键盘） |
| ↑ / ↓ | 在消息列表中移动焦点（建议实现） |

## 移动端操作建议

### 手势
| 手势 | 操作 |
|------|------|
| 单指向上滑动（消息区） | 查看更早历史消息 |
| 单指向下滑动（消息区顶部） | 下拉刷新（如实现） |
| 长按消息气泡 | 复制/删除/转发上下文菜单 |
| 点击重试按钮 | 手动触发重连 |
| 点击时钟角标 | 查看暂存消息列表（建议实现） |

### 减少动效模式（Reduce Motion）
| 条件 | 处理方式 |
|------|---------|
| `prefers-reduced-motion: reduce` | 关闭所有入场/出场动画（直接切换 opacity，无位移） |
| 仍需保留 | 静态 opacity 变化（0.92 暗淡、0→1 淡入），无时长要求但应保留语义 |

### 动态字体（Dynamic Type）
| 场景 | 建议 |
|------|------|
| 用户设置超大字体 | 气泡最大宽度自适应，不固定 250 pt（改用百分比或最大屏宽 80%） |
| 横幅文字超长 | 允许换行，横幅高度自动增加 |
| 重试按钮文字截断 | 不允许截断，按钮宽度随文字增加 |

## 已知无障碍风险项

| 风险 | 严重程度 | 建议修复 |
|------|---------|---------|
| 重试按钮颜色对比度 1.8:1 | 高 | 加深至 #E8799A 或加深背景色 |
| 时间戳对比度 2.8:1 | 中 | 加深至 #868686 |
| Nav 副标题对比度 3.2:1 | 低（可接受，大字） | 监控，若用户反馈再调整 |
| 时钟按钮 36pt 小于 44pt 触摸目标 | 高 | 扩大 hitSlop 8pt 各方向 |
