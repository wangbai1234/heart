# 02 Layout — 离线降级状态页

## 画布规格
| 属性 | 值 |
|------|----|
| 逻辑分辨率 | 390 × 844 pt（iPhone 14） |
| 画布尺寸 | 1024 × 1536 px（@2.63x 倍率） |
| Safe Area Top | 47 pt（含状态栏 + 导航栏） |
| Safe Area Bottom | 34 pt（Home Indicator） |
| 背景色 | #FFF0F0 → #FFF8F3（从上至下渐变，Surface Cream） |

## 滚动区域
- **可滚动区域**：[D] 历史消息区（从横幅底部到 Composer 顶部）
- **固定不动**：[A] 状态栏、[B] 导航栏、[C] 横幅、[F] Composer、[G] Home Indicator
- **滚动方向**：纵向，向上滚动查看更多历史消息
- **弹性效果**：iOS 系统 rubber-band 滚动
