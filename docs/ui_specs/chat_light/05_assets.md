# 05 Assets — 聊天页 Chat（浅色模式）

本文件覆盖 Chat Light 的资源真相。旧版“程序生成整页背景”的说法失效。

## 1. 必须引用的复杂资源

| 用途 | 绝对路径 | 使用方式 |
|---|---|---|
| 全屏浅色背景 | `/Users/wanglixun/heart/assets/backgrounds/亮色背景图.png` | 全屏 `cover`，锚点 `center center` |

## 2. 背景使用规则

- Chat Light 的整页背景必须直接使用 `亮色背景图.png`
- 不允许改成代码渐变版
- 不允许重新生成云层、星点、暖光
- 允许在其上叠加轻微玻璃层、气泡、输入栏，但不得破坏背景图主观感受

## 3. 头像规则

- 本页设计稿展示的是运行态角色头像
- 若运行时已有真实角色头像，必须用真实角色头像
- 若运行时暂无绑定头像，必须使用“空白占位头像组件”
- 空白占位头像不得使用神无月凛、桃乐丝或任何角色图作为兜底
- 不允许 AI 生成“小屿”替代头像

## 4. 必须由组件实现的元素

- Header
- 返回按钮
- 更多按钮
- AI / User Bubble
- Voice Bubble
- Typing Indicator
- Composer
- Play / Send / Plus 图标
- 波形可视化

## 5. 禁止事项

- 禁止把整页背景改成 CSS 渐变
- 禁止新画聊天背景
- 禁止用图片代替气泡、Composer、波形
