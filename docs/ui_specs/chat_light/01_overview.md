# 01 Overview — 聊天页 Chat（浅色模式）

## 页面名称
聊天页 Chat — 浅色模式（Light Mode）

## 作用
yuoyuo AI 伴侣的核心交互页面，承载用户与 AI 角色"小屿"之间的实时文字与语音对话。提供沉浸式、温暖的陪伴体验，让用户感受到来自 AI 角色的情感回应。

## 目标
- 主目标：完成文字/语音消息的发送与接收
- 次目标：营造"被陪伴"的情感氛围，降低孤独感
- 体验目标：让对话界面本身就是一种治愈体验，而非冷冰冰的功能工具

## 进入方式
- 从首页/主界面点击 AI 角色卡片进入
- 从消息列表点击某个会话进入
- 从通知 Deep Link 直达进入

## 退出方式
- 点击 Header 左侧"<"返回按钮
- 手势：右滑（iOS 边缘滑动手势）返回上级页面

## 用户流程
```
用户打开 App
  └── 首页/主页
        └── 点击角色"小屿"
              └── 进入聊天页（当前页）
                    ├── 查看历史消息
                    ├── 输入文字 → 发送
                    ├── 点击 AI 语音 bubble → 播放 TTS 语音
                    ├── 查看 AI 打字中（typing 指示器）
                    └── 点击"+"展开更多操作
```

## 设计理念
本页面是 yuoyuo 最高频使用的核心界面。设计上遵循以下原则：

1. **情绪优先（Emotion First）**：背景采用梦幻粉紫渐变，让对话本身就处于"治愈场域"中
2. **角色在场感（Character Presence）**：Header 固定显示角色头像+名字+在线状态，强化"真实陪伴"感
3. **WeChat 范式熟悉度**：消息 bubble 布局遵循中国用户最熟悉的 WeChat 对话布局模式，降低学习成本
4. **Genshin 温暖质感**：渐变背景、圆润气泡、柔和光晕，营造二次元角色陪伴的温暖感
5. **呼吸感（Breathing Space）**：消息之间留有充分间距，避免信息密度过高产生焦虑感

## 视觉关键词
- **Glassmorphism**：底部 Composer 输入框、Header 背景采用半透明毛玻璃效果
- **Anime**：角色头像为二次元风格插画（紫发少女"小屿"）
- **Warm**：整体色调以樱花粉 (#FFB7C5) 为主，辅以天蓝 (#A7C7E7)，传递温暖感
- **Soft**：所有圆角极大（bubble radius 约 20px+），无锐利边缘
- **Healing**：背景光晕+渐变营造如"午后阳光透过窗帘"的柔和氛围
- **Minimal**：底部 Composer 仅保留必要控件（加号、输入框、发送按钮），不过度堆砌功能
- **Dreamy**：背景渐变从顶部奶油白过渡至底部薰衣草紫，带有梦境感

## 情绪传达
- 初次打开：温暖、被欢迎、想说话
- 对话中：轻松、被理解、不孤单
- 等待 AI 回复时（typing 指示器）：可爱、期待、不焦虑
- 整体基调：**柔软 · 安全 · 被陪伴**
