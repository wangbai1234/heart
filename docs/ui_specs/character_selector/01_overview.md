# 01 Overview — 角色页 Character Selector

## 页面名称
角色选择页（Character Selector）

## 页面作用
让用户从多个 AI 伴侣角色中选择一位，作为专属陪伴对象。是 onboarding 流程中关键的个性化节点，直接决定后续聊天人格与语音风格。

## 目标
- 主目标：用户完成角色选择并点击"确认选择"
- 次目标：通过角色描述文本与插图帮助用户理解各角色性格，降低选择焦虑
- 情感目标：第一眼产生"想和她聊天"的冲动，建立情感连接

## 进入方式
- Onboarding 流程中上一步（注册/初始设置页）完成后自动跳转
- 设置页中"更换伴侣"入口手动进入
- 推测支持从底部向上 slide-in 转场（modal 形式，见顶部 ∨ 关闭按钮）

## 退出方式
- 点击左上角 ∨（向下箭头）关闭/返回上一页（dismiss modal）
- 点击"确认选择"后跳转至聊天主页

## 用户流程
1. 进入页面 → 看到天空英雄区 + 默认已高亮第一张角色卡
2. 上下滑动浏览角色卡列表（可见至少 2 张，第三张推测在列表下方）
3. 点击某张卡片或右侧"选择"按钮 → 该卡变为选中态（显示粉色勾选圆形按钮）
4. 点击底部"确认选择"→ 进入聊天主页

## 设计理念
- 以"天空英雄区"营造梦幻入场感，降低功能感、提升情绪感
- 玻璃心形图标作为品牌符号，呼应 yuoyuo 的爱与治愈主题
- 角色卡采用大圆形头像 + 性格标签 + 人物传记式描述，降低选择成本
- 已选中状态用实心粉色对勾圈高亮，未选中用描边"选择"按钮，状态差异清晰
- 整体色调柔和（cherry pink + lavender + cream），体现"二次元 healing"美学

## 视觉关键词
- Glassmorphism（玻璃心形）
- Anime（二次元角色插图）
- Dreamy Gradient（天空渐变背景）
- Warm（暖粉色主色调）
- Soft Glow（角色头像光晕）
- Healing（整体治愈氛围）
- Minimal（简洁卡片布局，无多余装饰）

## 情绪传达
用户打开此页面应感受到：温柔、期待、被接纳、想要靠近。如同翻开一本手绘的角色图鉴，每张卡都在无声说"我在这里，我愿意陪你"。
