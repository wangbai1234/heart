# 05 资源规范（Assets）

本模块是动效分镜参考，不是资源生产清单。旧版“需生成”的表述失效。

## 1. 资源定位

- `motion_storyboard` 用来定义动效对象，不新增独立图片资源
- 动效承载对象必须优先复用现有页面组件与本地资源

## 2. 允许复用的既有资源

| 用途 | 绝对路径 | 使用方式 |
|---|---|---|
| 浅色背景参考 | `/Users/wanglixun/heart/assets/backgrounds/亮色背景图.png` | 作为浅色分镜底图 |
| 深色背景参考 | `/Users/wanglixun/heart/assets/backgrounds/暗色背景图.png` | 作为深色分镜底图 |

## 3. 动效对象规则

- Emotion Orb：必须作为组件实现，不新增图片资产
- Avatar：使用真实头像或空白占位头像组件，不新增 `avatar_youyou_circle@*.png`
- Voice Wave：必须作为代码组件实现
- 返回、更多、麦克风、加号：使用现有图标系统

## 4. 文档属性

- 分镜箭头、序号圆圈属于文档标注元素，不进入运行时代码资源清单
- 不允许因为分镜文档而为工程新增无业务价值的图片文件

## 5. 禁止事项

- 禁止生成新的 Emotion Orb 图片资源
- 禁止生成新的演示头像资源
- 禁止把语音波形做成静态图片
