## 帧④（页面过渡）特殊布局

帧④展示两个层叠状态：
- **底层（首页列表）**：向下位移 8px，亮度降至 92%，scale 0.99
- **顶层（聊天页 Sheet）**：从底部向上滑入，覆盖首页

```
[底层首页]  z-index: 0  opacity: 0.92  transform: translateY(8px) scale(0.99)
[顶层聊天] z-index: 10  transform: translateY(0) — 滑入完成状态
```

## 消息气泡布局

### AI 消息气泡（左对齐）

```
[头像 40px] [气泡内容]
                [时间戳]
```

- 头像圆形，直径约 40px
- 气泡距头像约 8px
- 气泡内 Padding：约 12px 水平 / 10px 垂直（估算值）
- 时间戳右下对齐，字号约 11px

### 用户消息气泡（右对齐）

```
            [气泡内容] [已送达标志 ✓]
                       [时间戳]
```

- 气泡背景：cherry pink #FFB7C5（浅色版本）
- 时间戳含 ✓ 符号，右对齐
