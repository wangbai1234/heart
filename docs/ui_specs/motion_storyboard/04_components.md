## 5. 情绪球 `EmotionOrb`（核心动效组件）

| 属性 | 值 |
|------|----|
| 作用 | 导航栏中央 AI 状态可视化，呼吸脉冲动效 |
| 尺寸 | 帧①③：约 36px 直径（正常态，估算值） |
|  | 帧②：约 80px 直径（放大强调状态，估算值） |
| 形态 | 圆形，玻璃态渐变球体 |
| 渐变 | 核心白色→天空蓝→薰衣草紫→樱花粉，径向渐变 |
| 外发光 | 多色光晕（紫/蓝/粉），blur 约 20px |
| 动效 | scale 1.0 → 1.06 → 1.0（往复），光晕 opacity 渐变 |
| 周期 | 1200ms，ease-in-out，无限循环 |
| 可复用性 | 是，核心组件，设置 `size`/`animating` prop |

**状态枚举**：
- `idle`：正常呼吸脉冲（scale 1.0↔1.06）
- `speaking`：说话时加强光晕（设计稿未定义详细状态）
- `listening`：（设计稿未定义）

## 可复用组件优先级

| 优先级 | 组件 | 原因 |
|--------|------|------|
| P0 必须复用 | `EmotionOrb`、`InputBar`、`AIMessageBubble`、`UserMessageBubble` | 所有聊天页必用 |
| P1 应复用 | `ChatNavBar`、`AIAvatar`、`StatusBar`、`VoiceWave` | 多页面共用 |
| P2 按需复用 | `HomeConversationList` | 首页专属 |
| 文档专用 | `DeviceFrame`、`StoryboardArrow`、`MotionAnnotation` | 不进入运行时 |
