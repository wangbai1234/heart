# 06 交互规范（Interactions）

## 说明

本文件基于分镜图中可见的 4 帧动效，结合 yuoyuo 设计语言，详细定义每条动效的交互参数。设计稿未明确定义的交互，统一标注为"**设计稿未定义**"。

---

## 动效 ① — 气泡绽放（Bubble Bloom）

### 触发条件
- 新消息气泡出现时（AI 回复 或 用户发送后本地即时呈现）

### 动效参数

| 属性 | 起始值 | 结束值 | 时长 | 缓动 |
|------|--------|--------|------|------|
| `scale` | 0.92 | 1.0 | 300ms | ease-out |
| `opacity` | 0 | 1.0 | 300ms | ease-out |
| 粉色光晕 `opacity` | 1.0 | 0 | 150ms（同时发生） | ease-out |

### 具体描述
1. 新消息气泡从 scale 0.92 缩放至 1.0，同时 opacity 从 0 变为 1.0
2. 气泡出现的同时，伴随粉色光晕（cherry pink glow）从气泡中心向外散射，然后在 150ms 内淡出消失
3. 整体动效总时长：300ms
4. 缓动类型：ease-out（快进慢出，模拟弹出感）

### 进入动画
- 触发方：新消息到达
- 动效类型：scale + opacity + glow

### 退出动画
- **设计稿未定义**（气泡出现后保持静止，无退出动效）

### 错误状态
- **设计稿未定义**（消息发送失败气泡样式未定义）

### 加载/Skeleton 状态
- **设计稿未定义**（AI 思考中气泡 loading 状态未定义）

## 动效 ② — 情绪球脉冲（Emotion Orb Pulse）

### 触发条件
- 进入聊天页后立即启动，持续无限循环

### 动效参数

| 属性 | 起始值 | 峰值 | 结束值 | 周期 | 缓动 |
|------|--------|------|--------|------|------|
| `scale` | 1.0 | 1.06 | 1.0 | 1200ms | ease-in-out |
| 光晕 `opacity` | 较弱 | 较强 | 较弱 | 1200ms | ease-in-out |

### 具体描述
1. 情绪球从 scale 1.0 缓慢扩大至 1.06，再缓慢收缩回 1.0
2. 光晕亮度/不透明度同步变化：scale 最大时光晕最亮，scale 最小时光晕最暗
3. 周期 1200ms，ease-in-out（模拟生物呼吸节律）
4. 无限循环（infinite），无停止条件（退出聊天页时停止）

### 进入动画
- 进入聊天页时情绪球淡入并开始脉冲（**设计稿未定义淡入细节**）

### 退出动画
- 离开聊天页时情绪球淡出（**设计稿未定义退出细节**）

### 暂停条件
- **设计稿未定义**（是否在用户打字时暂停？）

### 不同情绪状态
- **设计稿未定义**（不同情绪下是否改变颜色或频率？）

## 动效 ③ — 语音波（Voice Wave）

### 触发条件
- AI 语音（TTS）开始播放时触发
- 随音频实时驱动，无固定时长

### 动效参数

| 属性 | 描述 |
|------|------|
| 驱动方式 | 实时（音频频率/振幅分析） |
| 点数量 | 5 个 |
| 点颜色 | 左粉色 #FFB7C5 → 右薰衣草 #C8B6FF |
| 振幅范围 | 约 4px（静默）— 约 24px（峰值，估算值） |
| 缓动 | linear（实时映射） |
| 结束方式 | 语音停止后波形收缩回静默状态 |

### 具体描述
1. AI 语音播放时，输入栏上方出现 5 个圆形波点
2. 5 个波点高度随当前音频振幅实时变化
3. 颜色从左到右由粉色渐变到淡紫色
4. 中心点通常振幅最高（主频集中区）
5. 语音结束后波点渐回基准高度并淡出（**设计稿未定义淡出动效**）

### 进入动画
- 语音开始时波点淡入（**设计稿未定义具体淡入参数**）

### 退出动画
- **设计稿未定义**（建议：300ms ease-out 淡出并收缩）

### 静默状态
- 波点保持最小高度（约 4px）轻微跳动，或完全不显示（**设计稿未定义**）

### Hover / Pressed 状态
- **设计稿未定义**（语音波是否可点击暂停？）

## 动效 ④ — 页面过渡（Page Transition / Apple Sheet 风格）

### 触发条件
- 从首页对话列表点击某个对话条目

### 动效参数

| 层级 | 属性 | 起始值 | 结束值 | 时长 | 缓动 |
|------|------|--------|--------|------|------|
| 底层（首页） | `translateY` | 0 | +8px（向下） | 400ms | ease-in-out |
| 底层（首页） | `scale` | 1.0 | 0.99 | 400ms | ease-in-out |
| 底层（首页） | `brightness` | 1.0 | 0.92 | 400ms | ease-in-out |
| 顶层（聊天页） | `translateY` | 100vh（屏幕底部） | 0 | 400ms | Apple Spring |
| 轻微拖拽提示 | 底部出现 | — | — | 动效开始前 | — |

### 具体描述
1. 用户点击对话条目后，聊天页从屏幕底部以 Sheet 风格向上滑入
2. 同时，底层首页轻微向下移动 8px、轻微缩小至 0.99、亮度降至 92%
3. 顶层聊天页完全覆盖底层首页
4. 底部出现轻微拖拽提示（拖拽手柄 handle indicator）
5. 总时长 400ms，使用 Apple 系统 Spring 曲线

### 进入动画（聊天页）
- 从 translateY(100%) → translateY(0)，时长 400ms，Apple Spring

### 退出动画（返回首页）
- **设计稿未定义**（建议：反向：聊天页向下滑出，首页恢复 translateY(0) scale(1.0) brightness(1.0)）

### Dismiss 手势
- **设计稿未定义**（是否支持下滑手势关闭聊天页？）

### 中断状态
- **设计稿未定义**（动效进行中是否可中断？）

## 可访问性相关交互

### Reduced Motion
- 当用户开启"减少动态效果"时：
  - 气泡绽放：仅 opacity 0→1，取消 scale 变化
  - 情绪球：停止脉冲，保持静止
  - 语音波：简化为静态进度指示器
  - 页面过渡：使用 cross-fade 替代滑动
- **设计稿未定义 Reduced Motion 版本，以上为建议规范**
