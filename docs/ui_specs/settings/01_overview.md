# 01 Overview — 设置页 Settings

## 页面名称
设置页（Settings）

## 作用
提供用户账号信息展示、会员状态管理、外观偏好设置、通知偏好、隐私与数据管理、以及关于应用的汇总入口。是 App 的全局配置中心。

## 目标
- 让用户快速了解自己的账号与会员状态
- 提供主题/字体大小等个性化外观控制
- 管理通知偏好（积极提醒 + 静音时段）
- 支持隐私操作（清除数据、导出数据、注销账号）
- 提供应用信息与法律文件入口

## 进入方式
- 从底部 Tab 栏「我」（个人中心）页面进入
- 或从其他页面的快捷入口跳转（估算，具体流程见产品文档）

## 退出方式
- 点击左上角返回箭头 `<` 回到上一页（个人中心）
- iOS 左滑手势返回

## 用户流程
```
个人中心页
  └─ 点击「设置」入口
       └─ 设置页（当前页）
            ├─ 点击 Profile 卡 → 账号编辑页
            ├─ 会员兑换 → 展开/收起子选项（下拉/折叠）
            ├─ 订阅状态 → 订阅详情页
            ├─ 主题切换（浅色/深色/自动）→ 即时生效
            ├─ 字体大小 Slider → 即时预览
            ├─ 积极提醒 Toggle → 开/关通知
            ├─ 静音 → 静音时段设置页
            ├─ 清除网格服务器 → 确认弹窗
            ├─ 导出我的数据 → 导出流程页
            ├─ 注销账号 → 危险操作确认弹窗
            ├─ 版本 1.0.0 → （通常只读，可能隐藏开发者选项）
            ├─ 用户协议 / 隐私政策 → WebView
            └─ 联系我们 → 客服页 / WebView
```

## 设计理念
- 继承 yuoyuo 整体视觉语言：柔和粉紫调渐变背景 + 白色/浅色卡片
- 使用分组卡片（Card Group）而非传统 TableView 行列，视觉更现代
- 分组标题（Section Label）使用浅灰色小字，低视觉权重，不抢主角
- Profile 卡使用动漫角色头像强化 IP 感
- 危险操作（注销账号）使用粉红警示色，不使用红色，与整体色调协调
- Toggle/Switch/Slider 等交互控件保持品牌粉色调

## 视觉关键词
- Glassmorphism（轻度，卡片白色半透明）
- Anime / 二次元（角色头像插画）
- Warm（暖粉色调背景）
- Soft（柔和渐变，无硬边界）
- Healing（治愈感，紫色光晕背景装饰）
- Minimal（内容密度适中，克制不繁琐）
- Organized（分组清晰，视觉层次鲜明）

## 情绪传达
用户打开设置页时感受到：
> "这是我的小窝，整洁、温柔、有秩序感。yuoyuo 懂我，给我最合适的配置选项。"

整体基调：**安心、可控、温暖、亲切**。没有冷漠的系统感，即使是危险的「注销账号」也以品牌粉色柔化处理，降低焦虑感。
