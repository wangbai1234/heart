## 页面作用

本看板是 yuoyuo AI伴侣应用的**文字系统定义文档**，用于统一整个产品的字体使用规范，包含：

1. **6级字号阶梯**：从 Display（40px）到 Tabular（14px）的完整尺寸体系
2. **字体家族说明**：中文、拉丁、数字三类字体的选型依据
3. **实际应用范例**：浅色模式与深色模式下的聊天气泡段落效果
4. **颜色层级说明**：不同语义文本的颜色对应关系（如 Caption 的 #8A8A98）

## 页面目标

- 为设计师提供单一可信来源（Single Source of Truth）的字体规范
- 为工程师提供精确的 fontSize / lineHeight / fontWeight / fontFamily 参数
- 确保中文与拉丁字符在视觉上协调统一
- 验证字体在浅色 / 深色两种模式下均具备良好可读性

## 设计理念

### 核心原则

yuoyuo 的字体系统基于以下三条核心原则：

**1. 中文优先，中拉和谐**
- 中文字体选用 PingFang SC / HarmonyOS Sans SC，笔形圆润、几何感强、气质柔和
- 拉丁字符配合 SF Pro Rounded，圆角末端与中文字形气质呼应
- 两者在同一字符串中（如「你好，yuoyuo」）视觉重量协调，无突兀感

**2. 尺寸阶梯清晰，层级秩序明确**
- 6个字号从 40px 到 13px 形成显著的视觉层级差
- 行高比例（fontSize:lineHeight）在 1.2~1.4 之间，保证段落的呼吸感
- Display 级别承担情感引导，Body 级别承担信息传递，Caption 级别承担辅助说明

**3. 数字单独处理，等宽保持整齐**
- 时间戳（12:34）、时长（32m）等数字使用 SF Pro Rounded Tabular
- 等宽数字确保聊天记录中的时间列对齐整洁

### 情感定位

字体系统传达出「**温柔、清晰、亲密、可信**」的情绪氛围：
- 圆润字形 → 削弱冷硬感，强化陪伴温度
- 适中字重（Regular / Medium 为主）→ 非强迫性阅读，轻松自然
- SemiBold 仅用于 Display 级别 → 在情感表达时赋予力量，但不压迫

## 视觉关键词

| 关键词 | 体现方式 |
|--------|----------|
| Warm（温暖） | 圆润字形、中文柔和线条 |
| Healing（治愈） | 适中字重、舒适行高 |
| Minimal（简洁） | 6级有限阶梯，避免字号碎片化 |
| Readable（可读） | 中英文协调配对，行高充足 |
| Anime（动漫） | 字体气质偏日系二次元清新风 |
| Soft（柔软） | PingFang SC 笔形无硬角 |

## 情绪传达

用户（产品消费者）在 yuoyuo 中读到的每一行文字，都应该感受到：

> 「这是一个温柔的朋友在说话，不是机器在输出信息。」

字体系统从根本上支撑这个情绪传达——圆润不尖锐，亲密不冷漠，清晰不凌乱。
