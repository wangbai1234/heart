## 整体布局结构

```
┌──────────────────────────────────────────────────────────────────────────┐
│  Header Bar（标题行）                                                      │
│  "yuoyuo · 字体比例"                                                       │
├─────────────────────────────────┬────────────────────────────────────────┤
│  Left Panel（字级阶梯区）          │  Right Panel（应用示例区）               │
│  ~600 px wide                    │  ~800 px wide（估算值）                  │
│                                  │  ┌──────────────────────────────────┐  │
│  [Display]  你好，yuoyuo          │  │ 浅色模式示例（Light Mode Card）   │  │
│  [Title]    《今天聊点什么》        │  └──────────────────────────────────┘  │
│  [Headline] 《晚安，做好梦》       │  ┌──────────────────────────────────┐  │
│  [Body]     我在这里陪你。         │  │ 深色模式预览（Dark Mode Card）    │  │
│  [Caption]  刚刚 · 已读           │  └──────────────────────────────────┘  │
│  [Tabular]  12:34 · 32m          │                                          │
├─────────────────────────────────┴────────────────────────────────────────┤
│  Footer Bar（字体家族注释区）                                               │
│  "中文字体 / 拉丁字体 / 数字字体" 说明 + Aa 图标 + yuoyuo logo             │
└──────────────────────────────────────────────────────────────────────────┘
```

#### 字级行内布局

每一行字级由三列组成：

```
[级别标签列]  [圆点 ●]  [示例文字 + 字体注释]
```

| 列 | 宽度（估算值） | 内容 | 对齐 |
|----|----------------|------|------|
| 级别标签列 | 约 100 px | "Display / 40/48" | 右对齐，垂直居顶 |
| 圆点列 | 约 24 px | 粉色实心圆 ● | 水平居中 |
| 示例 + 注释 | 剩余宽度 | 示例文字（大）+ 字体注释（小，灰） | 左对齐 |

### Right Panel — 应用示例区

## 对齐方式

| 对齐关系 | 方式 |
|----------|------|
| 整体左右 | 两列 Flex Row，等高对齐顶部 |
| 字级标签 | 右对齐至圆点列 |
| 示例文字 | 左对齐，垂直居中于字级行 |
| 右侧卡片 | 左对齐至 Right Panel 左边缘 |
| Footer 文字 | 左对齐；品牌字右对齐 |
