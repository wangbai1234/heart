## 资源命名规范

```
图标类：    icon_{name}.svg
头像类：    avatar_{角色}_{主题}.png
贴图类：    sticker_{name}.png
背景类：    bg_{name}.{png|svg}
插画类：    illust_{场景}_{尺寸}.png
```

## 图片尺寸适配建议

| 场景 | 倍率 | 说明 |
|------|------|------|
| 头像（44 px 显示） | @1x = 44 px | — |
| | @2x = 88 px | Retina 标准 |
| | @3x = 132 px | iPhone ProMax 等 |
| 角色插画（大） | @1x + @2x | 按设备 DPR 加载 |
| SVG 资源 | 矢量，任意尺寸 | 优先用 SVG |
