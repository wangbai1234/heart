# 04 组件分析 — Typography Specimen（字样规范）

## 组件总览

本看板共包含以下组件类型：

1. 看板标题行（Board Header）
2. 字级行（Type Scale Row） × 6
3. 字体族注释区（Font Family Footer）
4. 浅色模式聊天预览卡片（Light Mode Chat Card）
5. 深色模式聊天预览卡片（Dark Mode Chat Card）
6. 聊天气泡（Chat Bubble） — AI方 / 用户方 × 2 套配色
7. 聊天头像（Avatar）
8. 状态标签行（Status Bar Row）— 卡片顶部/底部
9. Aa 装饰图标（Font Icon Badge）
10. 品牌字（Brand Wordmark）

---

## 1. 看板标题行（Board Header）

| 属性 | 值 |
|------|----|
| 组件名 | `BoardHeader` |
| 作用 | 声明本看板类型和所属产品 |
| 层级 | 画布顶层，固定位置 |
| 尺寸 | 全宽 × 约 40 px 高（估算值） |
| 布局 | Flex Row，左对齐 |
| 内容 | 文字：「yuoyuo · 字体比例」 |
| 字体 | PingFang SC SemiBold，约 24 px |
| 颜色 | `#3A3A4A` |
| 状态 | 静态，无交互 |
| 可复用性 | 中等；作为设计文档专用组件，不进入产品代码 |

## 2. 字级行（Type Scale Row）

### 通用结构

每个字级行是看板的核心展示单元，结构统一：

```
[级别标签]  [粉色圆点]  [示例文字]
                        [字体注释小字]
```

| 属性 | 值 |
|------|----|
| 组件名 | `TypeScaleRow` |
| 作用 | 展示单个字号级别的字体尺寸、字重、字体族 |
| 层级 | Left Panel 内，Flex Column 排列 |
| 布局 | Flex Row，左右列对齐 |
| 状态 | 静态 |
| 可复用性 | 高（设计文档内可复用，作为 token 展示模板） |

## 3. 字体族注释区（Font Family Footer）

| 属性 | 值 |
|------|----|
| 组件名 | `FontFamilyFooter` |
| 作用 | 解释三类字体的选型依据，供开发者参考 |
| 层级 | 画布底部，Footer 区域 |

| | 「拉丁字体：SF Pro Rounded（圆润、友好、可读性高）」 |
| | 「数字字体：SF Pro Rounded Tabular（等宽数字，便于阅读时间与数据）」 |
| 注释字号 | 约 13 px（估算值） |
| 注释颜色 | `#6B6B7E`（估算值，中等灰） |
| 状态 | 静态 |
| 可复用性 | 低（设计文档专属，不进入产品代码） |
