## 说明

### 为何本资产所有交互均不适用

本 PNG 文件（`03_typography.png`）是一个**设计规范展示看板**（Design Spec Board），其目的是：

1. 向设计团队和开发团队传达字体系统定义
2. 以可视化方式展示6个字号级别及其属性
3. 通过聊天气泡模拟图验证字体在实际场景中的效果

看板本身不是一个可操作的产品界面（UI Screen），因此不存在用户交互路径。

| 交互类型 | 触发方式 | 预期行为 |

## 文字渲染相关的交互注意事项

虽非直接交互，以下内容影响字体在动态场景中的表现：

### 文字截断规则

| 场景 | 处理方式 | 建议 |
|------|----------|------|
| 聊天气泡过长 | 换行（不截断） | 气泡最大宽度约 70% 屏宽 |
| 用户名过长 | 末尾省略（…） | 单行显示，超出截断 |
| 时间戳 | 不换行 | 固定格式，等宽字体保障对齐 |
| 已读状态文字 | 不换行 | 单行显示 |

### 字体加载策略

| 字体 | 加载方式 | 备注 |
|------|----------|------|
| PingFang SC | iOS 系统字体，无需加载 | 非 iOS 设备需 fallback 到 HarmonyOS Sans SC 或 Noto Sans SC |
| SF Pro Rounded | iOS 系统字体，无需加载 | 非 iOS 设备需 fallback 到 Inter Rounded 或 Nunito |
| HarmonyOS Sans SC | 需嵌入 / CDN 加载（Android / Web） | 加载期间使用系统 sans-serif 做 fallback |

### 字体 Fallback 链

```
中文：
"PingFang SC", "HarmonyOS Sans SC", "Noto Sans SC", sans-serif

拉丁/英文：
"SF Pro Rounded", "Inter", "Nunito", sans-serif

数字（等宽）：
"SF Pro Rounded", "Inter", monospace
```
