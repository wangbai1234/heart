# 02 布局规范 — Icon Set（24×24 系统图标）

## 画布规格

| 属性 | 值 |
|------|-----|
| 画布尺寸 | 1024×1024 px |
| 画布比例 | 1:1 正方形 |
| 背景色 | 暖奶油白 #FAF0EC（估算值） |
| 图标目标尺寸 | 24×24 dp（设计规范标准），画布中实际渲染尺寸约 112×112 px（估算值） |
| 总图标数量 | 36 个 |
| 排列方式 | 6列 × 6行 均匀网格 |

## 背景层次分析

### Layer 1：画布背景
- 颜色：暖奶油白 #FAF0EC（估算值）
- 尺寸：1024×1024 px
- 无纹理，纯色平铺

### Layer 2：白色卡片区域（隐式分组容器）
- 可见到每行图标后方存在极淡的圆角矩形背景区块（仅 Row 1 有明显白色圆角卡片底，其余行较透明）
- Row 1 明显白色卡片：约 920×120 px，圆角约 20 px（估算值）
- Row 2–6 背景更为淡化或几乎透明，图标直接在奶油底色上渲染

### Layer 3：图标本体
- 纯描边，无填充
- 每个图标均渲染在各自单元格中央

## 网格系统

### 核心网格参数（估算值）

| 参数 | 估算值 |
|------|--------|
| 画布总宽 | 1024 px |
| 画布总高 | 1024 px |
| 左边距（Padding Left） | 约 60 px（估算值） |
| 右边距（Padding Right） | 约 60 px（估算值） |
| 顶部边距（Padding Top） | 约 60 px（估算值） |
| 底部边距（Padding Bottom） | 约 60 px（估算值） |
| 内容区宽度 | 约 904 px（估算值） |
| 内容区高度 | 约 904 px（估算值） |
| 列数 | 6 列 |
| 行数 | 6 行 |
| 列宽（含间距） | 约 150 px（估算值）|
| 行高（含间距） | 约 150 px（估算值） |
| 图标单元格尺寸 | 约 112×112 px（估算值）|
| 列间距（Column Gap） | 约 38 px（估算值） |
| 行间距（Row Gap） | 约 38 px（估算值） |

### 对齐方式
- 水平对齐：每行图标等距分布，整体居中对齐
- 垂直对齐：每列图标等距分布，整体居中对齐
- 图标在单元格内：水平 + 垂直双居中（`align-items: center; justify-content: center`）

## Safe Area

**N/A for this asset type** — 图标库画布，不涉及设备 Safe Area。

图标在应用中使用时需遵守：
- iOS Safe Area Top: 47 px
- iOS Safe Area Bottom: 34 px
- 触控目标最小尺寸：44×44 dp（图标实际 24×24，需外加 10dp 四周 padding 达到触控目标）

## Flex / Grid 实现参考（仅布局逻辑，非代码）

- 外容器：正方形，居中对齐
- 网格类型：CSS Grid，6列固定，行高自适应等比
- 单元格对齐：`place-items: center`（双轴居中）
- 行列间距统一：`gap: 38px`（估算值）
- 外边距：`padding: 60px`（估算值）
