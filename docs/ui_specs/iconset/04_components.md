# 04 组件规范 — Icon Set（24×24 系统图标）

## 组件架构概述

图标库分为两个层次：
1. **原子组件**：单个图标（Icon Atom），SVG/矢量路径本体
2. **分子组件**：图标 + 触控区域 + 状态管理（IconButton / NavItem 等）

---

## 原子组件：单图标（Icon）

### 组件定义

| 属性 | 值 |
|------|-----|
| 组件名称 | `<Icon />` |
| 作用 | 渲染单个描边图标 SVG |
| 层级 | 原子级（Atom） |
| 尺寸规格 | 24×24 dp（标准），支持 16/20/24/32 dp 四档 |
| 描边宽度 | 1.5 dp（24dp尺寸基准） |
| 描边端点 | Round Cap |
| 描边连接 | Round Join |
| 填充 | None |
| 颜色 | 继承 `currentColor`，默认 `icon.default` (#4A4A6A) |

### Props 规范

| Prop | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `name` | string | — | 图标标识符（见下方命名规范） |
| `size` | 16\|20\|24\|32 | 24 | 图标尺寸（dp） |
| `color` | string | `currentColor` | 图标颜色 |
| `strokeWidth` | number | 1.5 | 描边宽度 |

## 分子组件：图标按钮（IconButton）

### 组件定义

| 属性 | 值 |
|------|-----|
| 组件名称 | `<IconButton />` |
| 作用 | 可交互图标，包含触控区域、状态反馈 |
| 层级 | 分子级（Molecule） |
| 触控区域 | 最小 44×44 dp（iOS HIG 规范） |
| 内部 Padding | 10 dp 四周均等（使 24dp 图标达到 44dp 触控区） |
| 背景 | 默认透明，Hover/Press 时显示圆角矩形背景 |
| 圆角 | 12 dp（触控背景圆角，估算值） |

### 状态规范

| 状态 | 图标色 | 背景色 | 缩放 |
|------|--------|--------|------|
| Default | `icon.default` (#4A4A6A) | transparent | 1.0 |
| Hover（Web端） | `color.primary` (#FFB7C5) | rgba(255,183,197,0.12) | 1.0 |
| Pressed | `icon.default` (#4A4A6A) | rgba(74,74,106,0.08) | 0.88 |
| Focused（键盘） | `icon.default` | rgba(74,74,106,0.08) + 2dp outline | 1.0 |
| Disabled | rgba(74,74,106,0.30) | transparent | 1.0 |
| Active/Selected | `color.primary` (#FFB7C5) | rgba(255,183,197,0.15) | 1.0 |

## 分子组件：底部导航项（NavItem）

### 组件定义

| 属性 | 值 |
|------|-----|
| 组件名称 | `<NavItem />` |
| 作用 | 底部导航栏单个标签项，图标+标签文字 |
| 层级 | 分子级（Molecule） |
| 布局 | 垂直堆叠：图标（24dp）上 + 文字（10px）下 |
| 总高度 | 约 48 dp（估算值） |
| 宽度 | 屏幕宽度 / 导航项数量 |
| 图标 | 选用本图标库对应图标 |
| 标签 | PingFang SC 10px，非选中=灰色，选中=primary |

### 使用的图标
- `home`（首页）
- `chat`（聊天）
- `ai-companion`（AI伴侣）
- `profile`（我的）

### 状态
| 状态 | 图标色 | 文字色 | Glow |
|------|--------|--------|------|
| 未选中 | #4A4A6A @ 60% | #4A4A6A @ 60% | 无 |
| 选中 | #FFB7C5 | #FFB7C5 | 粉色光晕 12px blur |

## 逐图标组件分析（36个）

### 导航类图标（Row 1）

#### 01 Home（房屋）
- 形态：简洁房屋轮廓，三角屋顶 + 方形门洞
- 尺寸：画布内约 80×80 px（估算值），24dp 规范尺寸
- 视觉特征：屋顶线条连续，门洞居中矩形，整体比例均衡
- 描边端点：Round Cap
- 复用建议：底部导航 NavItem，首页入口按钮
- 可复用：是

#### 02 Chat Bubble（聊天气泡+三点）
- 形态：圆角矩形气泡 + 底部角，内有三个等距圆点（···）
- 尺寸：气泡约 72×62 px（估算值），三点横向排列
- 视觉特征：气泡角朝左下，三点表示"输入中"或"消息"语义
- 描边端点：Round Cap
- 复用建议：底部导航聊天标签，消息通知入口
- 可复用：是

#### 03 AI Companion（人像+心形）
- 形态：标准人像圆形（头+肩轮廓）+ 右下角附加心形装饰
- 尺寸：人像圆约 64 px（估算值），心形约 22 px（估算值）
- 视觉特征：心形是 yuoyuo 独特设计亮点，区分普通用户图标
- 描边端点：Round Cap + Round Join
- 复用建议：AI伴侣关系页导航，好感度相关界面
- 可复用：是（差异化标志性图标）

#### 04 Settings（齿轮）
- 形态：标准8齿齿轮，中间圆孔
- 尺寸：约 78×78 px（估算值）
- 视觉特征：齿轮齿数均匀，中央圆孔尺寸适中，视觉稳定
- 描边端点：Round Cap
- 复用建议：设置页入口，系统配置相关
- 可复用：是

#### 05 Profile（人像圆形）
- 形态：人像轮廓（头+肩），与 AI Companion 相似但无心形
- 尺寸：约 70×70 px（估算值）
- 视觉特征：圆形外框包裹人像，纯描边
- 描边端点：Round Cap
- 复用建议：个人资料页，用户头像占位符
- 可复用：是

#### 06 Search（放大镜）
- 形态：圆形镜片 + 斜向握柄，经典放大镜构型
- 尺寸：镜片约 52 px（估算值），握柄约 28 px 斜线（估算值）
- 视觉特征：握柄朝右下，握柄端点圆头
- 描边端点：Round Cap
- 复用建议：搜索入口，全局搜索页
- 可复用：是

### 音频/媒体类图标（Row 3）

#### 13 Play（圆圈播放）
- 形态：圆形外框 + 内部朝右三角形播放符号
- 尺寸：外圆约 76 px（估算值），三角形略小于圆
- 视觉特征：三角向右居中，视觉重心稍偏右（补偿视觉不均）
- 描边端点：Round Join（三角角点）
- 复用建议：语音消息播放，TTS 播放按钮
- 可复用：是

#### 14 Pause（圆圈暂停）
- 形态：圆形外框 + 内部两条竖线（|| 暂停符）
- 尺寸：与 Play 相同外圆规格，双竖线间距适中
- 视觉特征：与 Play 形成视觉对，双竖线圆头端点
- 描边端点：Round Cap（竖线端点）
- 复用建议：语音播放中暂停状态（与 Play 切换）
- 可复用：是（与 Play 配对）

#### 15 Waveform（竖条波形）
- 形态：5–7条不等高竖线，居中高两侧低，模拟音频波形
- 尺寸：总宽约 80 px，中央竖线最高约 70 px（估算值）
- 视觉特征：动态感强，传递"录音中/音频活跃"状态
- 描边端点：Round Cap（每条竖线两端）
- 复用建议：录音激活状态指示，音频可视化
- 可复用：是（可动态变化高度做动效）

#### 16 Volume Up（扬声器+波纹）
- 形态：扬声器锥形 + 右侧两道弧形波纹
- 尺寸：约 80×64 px（估算值）
- 视觉特征：波纹弧线由内向外逐渐加大，传递"声音扩散"动感
- 描边端点：Round Cap（波纹端点）
- 复用建议：音量调节，语音输出激活
- 可复用：是

#### 17 Mute（扬声器+×）
- 形态：扬声器锥形（无波纹）+ 右侧×符号
- 尺寸：约 80×64 px（估算值）
- 视觉特征：×号替代波纹，语义清晰，与 Volume Up 形成对
- 描边端点：Round Cap
- 复用建议：静音状态，禁止音频输出
- 可复用：是（与 Volume Up 配对）

#### 18 Headphone（耳机）
- 形态：弧形头梁 + 两侧耳罩（圆形/方形）+ 耳垫
- 尺寸：约 80×72 px（估算值）
- 视觉特征：对称结构，头梁弧度自然，耳罩比例协调
- 描边端点：Round Cap + Round Join
- 复用建议：耳机模式，听声模式，音频设置
- 可复用：是

- 可复用：是（与 Arrow Left 配对）

#### 33 Close（×）
- 形态：两条对角线交叉，45°和135°方向
- 尺寸：约 64×64 px（估算值）
- 视觉特征：×号纯描边，线条均等，端点圆头
- 描边端点：Round Cap
- 复用建议：关闭弹窗，取消操作，删除标签
- 可复用：是

#### 34 Check（✓）
- 形态：L形钩子（短边朝左下，长边朝右上）
- 尺寸：约 70×54 px（估算值）
- 视觉特征：确认勾，比例自然，短边适中，圆头端点
- 描边端点：Round Cap + Round Join（折角）
- 复用建议：确认操作，完成状态，选中项，复选框
- 可复用：是

#### 35 More（···三点）
- 形态：三个等距圆点（实心圆/描边圆），水平排列
- 尺寸：三点总宽约 60 px，点径约 10 px（估算值）
- 视觉特征：经典"更多操作"通用图标，间距均等
- 描边端点：N/A（圆点）
- 复用建议：更多操作菜单，下拉操作，省略内容
- 可复用：是

#### 36 Trash（垃圾桶）
- 形态：矩形桶体 + 顶盖线 + 桶内三条竖线 + 盖上提手弧
- 尺寸：桶体约 56×62 px，提手约 40×14 px（估算值）
- 视觉特征：桶内三竖线细节丰富，提手弧度自然，传递"删除"语义
- 描边端点：Round Cap + Round Join
- 复用建议：删除消息，清空记录，移除操作
- 可复用：是
