## 设计理念

| 维度 | 说明 |
|------|------|
| 风格内核 | 二次元治愈美学（Healing Anime Aesthetic）结合现代极简线条 |
| 描边一致性 | 全套图标使用统一线宽（约 1.5px @24px，等比放大）保持视觉重量一致 |
| 圆润感 | 所有折角、端点均使用圆头（Round Cap）和圆角连接（Round Join），符合 yuoyuo 柔和亲切的产品气质 |
| 低噪声原则 | 纯描边无填充，与奶油色背景（#FFF8F3）形成低对比柔和搭配，避免视觉疲劳 |
| 语义清晰 | 每个图标保持国际通用可识别性，同时融入细节温度感（如AI伴侣图标带心形） |
| 色彩节制 | 图标默认使用单色墨水色（Charcoal #3A3A4A），激活/高亮状态可映射品牌色 |

## 情绪传达

整套图标传递的情绪为：**轻盈、温暖、亲切、值得信赖**。

描边而非填充的设计选择，使图标在视觉上"退后一步"，让内容和角色成为主角。圆润端点避免锐利感，与 yuoyuo 的治愈陪伴调性高度契合。图标整体视觉重量轻柔，适合在淡奶油色和毛玻璃背景上叠加，不会对界面产生压迫感。
