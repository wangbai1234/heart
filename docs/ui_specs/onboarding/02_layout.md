# 02 Layout — 首次引导 FirstVisitGuide

## 画布规格

| 属性 | 值 |
|------|----|
| 拼版画布尺寸 | 1536 × 1024 px |
| 单屏逻辑尺寸 | 390 × 844 px（iPhone 14） |

## 拼版画布布局

画布为横版三栏等分，每栏含一枚手机帧：

```
┌─────────────────────────────────────────────────────────────────┐
│  canvas: 1536×1024                                              │
│  background: #FEF0F0 (cream pink)                               │
│                                                                 │
│  ┌───────────┐  ┌───────────┐  ┌───────────┐                   │
│  │  Phone 1  │  │  Phone 2  │  │  Phone 3  │                   │
│  │ Step 1    │  │ Step 2    │  │ Step 3    │                   │
│  └───────────┘  └───────────┘  └───────────┘                   │
│                                                                 │
│  [步骤1—认识yuoyuo] [步骤2—你的隐私安全] [步骤3—解锁完整体验]      │
└─────────────────────────────────────────────────────────────────┘
```

| 属性 | 值（估算值） |
|------|-------------|
| 每栏宽度 | ~512 px（三等分 1536） |
| 手机帧宽度 | ~390 px |
| 手机帧高度 | ~780 px（在画布内，非完整844） |
| 手机帧水平居中 | 各栏中心对齐 |
| 手机帧顶部距画布顶 | ~40 px（估算值） |
| 底部标签区高度 | ~60 px（估算值） |
| 手机帧圆角 | ~40 px（估算值）|
| 帧间距 | ~16–24 px（估算值） |

## 单屏布局结构（以 Step 1 为基准，Step 2/3 同构）

### 层级结构

```
Page (390 × 844)
├── Layer 0: BackgroundGradient        z-index: 0
├── Layer 1: IllustrationArea          z-index: 1
├── Layer 2: ContentCard (白色毛玻璃)  z-index: 2
│   ├── TitleBlock
│   ├── SubtitleBlock
│   ├── PaginationDots
│   └── ActionButton(s)
└── Layer 3: StatusBar                z-index: 10
```

---

## 滚动区域说明

三步引导为**不可滚动页面**，每屏内容固定在视口内。切换屏幕通过：
- 点击"下一步"按钮
- 左右水平滑动（⚠️ 设计稿未明确定义，建议支持）

页面内部无竖向滚动。
