# 01 Overview — 首次引导 FirstVisitGuide（3屏拼版）

## 页面名称

首次引导 / FirstVisitGuide  
源文件：`16_onboarding_3up.png`  
拼版类型：横版 3-Up Mockup（三屏并排展示）

---

## 页面作用

在用户首次安装并打开 yuoyuo 后，依次展示三步引导页面，帮助用户：

1. 了解 yuoyuo 的核心定位（AI 伴侣身份）
2. 建立隐私信任（数据安全承诺）
3. 引导付费转化（会员解锁入口）

---

|--------|------|
| P0 | 在30秒内完成产品核心价值的情感传递 |
| P0 | 最小化用户认知摩擦，引导滑动到底 |
| P1 | 在第3屏完成首次付费意图触达 |

## 设计理念

**"第一印象即信任时刻"**

三屏引导遵循"情感共鸣 → 信任建立 → 行动转化"的心理路径：

- **Step 1**：以水晶心 + 梦幻云彩唤起情感共鸣，用"虚拟宇宙"建立角色身份认同
- **Step 2**：以玻璃罩小屋隐喻"你的秘密空间受到保护"，化解数据隐私焦虑
- **Step 3**：以礼盒 + 水晶心传达"加入即获得礼物"的仪式感，软化付费门槛

整体采用**渐进式信息密度**：插画占屏主导 → 文字辅助 → 按钮行动，视觉节奏平缓流畅。

## 视觉关键词

| 关键词 | 体现方式 |
|--------|----------|
| Glassmorphism | 水晶心的折射光感、玻璃罩吊坠的透明质感 |
| Anime / 二次元 | 插画风格（3D渲染+手绘感），柔软轮廓 |
| Warm / Healing | 粉色·薰衣草·奶油色调，云朵·小屋·礼盒意象 |
| Soft | 所有边缘均有羽化/渐变过渡，无硬边界 |
| Dreamy | 大面积朦胧光效，散射光斑（bokeh），梦境感背景 |
| Minimal | 每屏信息点≤3，界面元素克制，留白充足 |
| Playful | 水晶、吊坠、礼盒的少女系插画语言 |
