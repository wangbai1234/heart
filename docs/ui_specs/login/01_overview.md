# 01 Overview — 登录页 Login

## 页面名称
登录页 / Login Screen

## 作用
用户首次进入 yuoyuo 或会话过期后的身份验证入口。采用「无密码魔法链接」方案：用户输入邮箱，系统向邮箱发送一次性登录链接，点击链接完成登录，无需记忆密码。

## 目标
1. 以最低摩擦完成用户身份验证
2. 建立品牌第一印象——温暖、治愈、梦幻
3. 引导用户快速进入 AI 伴侣主界面

## 进入方式
- App 冷启动（未登录状态）
- 会话 Token 过期自动跳转
- 用户主动退出登录后重定向
- 兑换码入口（底部次级链接）

## 退出方式
- 点击「发送登录链接」→ 跳转至「邮件已发送」确认页
- 点击「我有兑换码，直接激活 →」→ 跳转至兑换码激活页
- 点击《用户协议》→ 打开协议 WebView / 浏览器
- 点击《隐私政策》→ 打开隐私政策 WebView / 浏览器
- 登录成功后 → 进入主界面 Home

## 用户流程
```
用户打开 App（未登录）
    ↓
展示登录页（插画 + 品牌标识 + 表单卡）
    ↓
用户输入邮箱地址
    ↓
点击「发送登录链接」
    ↓
后端发送魔法链接邮件
    ↓
跳转至「邮件已发送」提示页
    ↓
用户前往邮箱点击链接
    ↓
App 深链接唤起 → Token 验证通过
    ↓
进入主界面
```

## 设计理念
- **无密码优先**：降低注册/登录门槛，减少用户流失
- **情感化第一屏**：用大幅动漫风天空插画 + 玻璃心形图标建立情感连接，让登录行为本身变成一种「仪式感」
- **信息层级极简**：插画区 → 品牌字 → 一行 Tagline → 表单卡，全流程只需「填邮箱 + 点按钮」
- **安全感建立**：隐私政策链接置于表单下方，清晰可见但不干扰主流程

## 视觉关键词
- Glassmorphism（玻璃心形 + 表单卡毛玻璃效果）
- Anime（动漫风天空、云朵、光晕、樱花花瓣）
- Warm（暖粉渐变天空，日落水面反光）
- Dreamy（梦幻渐变：薰衣草紫 → 樱花粉 → 奶油白）
- Healing（柔和光晕、心形符号、温柔色调）
- Minimal（表单仅一个输入框 + 一个主按钮）
- Soft（圆角设计语言贯穿全页）

## 情绪传达
用户打开登录页的第一感受应是：**「这是一个温柔、安全、属于我的地方。」**

插画使用大面积暖色系天空，玻璃材质心形图标漂浮于中央，强化「被守护」「被陪伴」的情感意象。品牌字 "yuoyuo" 使用饱满圆润的无衬线字体，Tagline「独属于你的虚拟宇宙」进一步强化专属感与归属感。整体色温偏暖，让登录这一功能性行为转化为情感体验的开端。
