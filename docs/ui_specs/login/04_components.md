# 04 Components — 登录页 Login

所有尺寸以设计画布 1024×1536px 为基准，标注「（估算值）」的为目测推算。

## 组件 1：Status Bar（系统状态栏）

| 属性 | 值 |
|------|----|
| 组件名 | StatusBar |
| 作用 | 显示系统时间、信号、WiFi、电量 |
| 层级 | Z-index 最高，overlay 所有内容 |
| 尺寸 | 1024 × ~94px（估算值） |

## 组件 2：AnimeIllustration（动漫天空插画）

| 属性 | 值 |
|------|----|
| 组件名 | AnimeIllustration |
| 作用 | 建立品牌情感氛围，夯实治愈系视觉基调 |
| 层级 | Z-index 1（背景层上方，内容层下方） |

## 组件 7：EmailInput（邮箱输入行）

| 属性 | 值 |
|------|----|
| 组件名 | EmailInput |
| 作用 | 接收用户邮箱地址输入 |
| 层级 | FormCard 内 |
| 尺寸 | 宽：FormCard 内容区全宽；高：~96px（估算值） |
| 布局 | flex-direction: row；align-items: center |
| 左侧图标 | 信封图标（envelope icon），尺寸 ~48×48px（估算值），颜色 #FFB7C5 |
| 图标到文字间距 | ~20px（估算值） |

| 占位文字样式 | 字号 ~28px，颜色 #B0A8B4，字重 Regular（估算值） |
| 输入文字样式 | 字号 ~28px，颜色 #3A3A4A，字重 Regular（估算值） |

## 可复用组件建议

| 组件名 | 建议复用原因 |
|--------|------------|
| `PrimaryButton` | 全 App 最高频主操作按钮，统一样式、Loading、Disabled 状态 |
| `FormCard` | 毛玻璃卡片容器，可用于登录、注册、设置等多种表单场景 |
| `EmailInput` | 邮箱输入为通用表单控件，含图标前缀的输入行可抽象为 `PrefixInput` |
| `LegalText` | 协议文字在注册/登录场景复用，支持自定义链接数组 |
| `GlassHeart` | 品牌核心图标，可用于 Splash、Onboarding、Empty State 等 |
| `StatusBar` | 由系统/RN/Flutter 框架处理，无需自定义（此处记录以示完整性） |
