# 02 Layout — 登录页 Login

## 画布规格
- 逻辑分辨率：390 × 844 px（iPhone 14）
- 设计画布：1024 × 1536 px（2.626× 缩放比）
- 以下所有数值为设计画布尺寸（1024px 宽基准），标注「（估算值）」的为目测推算

---

## 层级结构（Z-index 由低到高）
| 层级 | 名称 | 说明 |
|------|------|------|
| 0 | Background Gradient | 全页渐变背景 |
| 1 | Illustration Layer | 动漫天空插画（上半区） |
| 2 | Content Layer | 品牌区 + 表单区 + 底部链接 |
| 3 | Form Card | 毛玻璃表单卡（叠于渐变背景之上） |
| 4 | Status Bar | 系统状态栏（最顶层） |

---

## 各区域详细规范

### Status Bar（状态栏）
- 宽：1024px（全宽）
- 高：~94px（估算值）
- 内容：左侧时间「9:41」，右侧信号+WiFi+电池图标
- 背景：透明（叠于插画之上）
- 文字色：#3A3A4A（深炭色）

### Illustration Zone（插画区）
- 宽：1024px（全宽，无水平边距）
- 高：~650px（估算值）
- 内容：全幅动漫风天空背景插画，玻璃心形图标居中偏上
- 插画与下方内容区的过渡：底部渐变淡出（插画底部 → 奶油白 #FFF8F3）
- 玻璃心形位置：水平居中，垂直约在插画区中央偏上（y≈320，估算值）
- 玻璃心形尺寸：约 280×280px（估算值）

### Brand Zone（品牌区）
- 宽：1024px（全宽）
- 高：~200px（估算值）
- 背景：#FFF8F3（奶油白）或与插画底部渐变平滑过渡
- 水平对齐：居中（center）
- 内部布局（垂直堆叠，flex-direction: column，align-items: center）：

  | 元素 | 字号（估算值） | 上边距 |
  |------|------------|------|
  | "yuoyuo" 字标 | ~96px | ~48px（距插画底部） |
  | Tagline「独属于你的虚拟宇宙」 | ~36px | ~20px（距字标底部） |

### Form Card（表单卡）
- 外层水平 Padding（距屏幕边缘）：~64px 左右各（估算值）
- 卡片宽：~896px（估算值，= 1024 - 64×2）
- 卡片高：~320px（估算值）
- 圆角：~48px（估算值）
- 背景：rgba(255,255,255,0.75) 毛玻璃效果，backdrop-blur
- 阴影：轻微投影（见 03_design_tokens.md）
- 内部 Padding：~48px 上下，~48px 左右（估算值）
- 内部布局（垂直堆叠）：

  | 元素 | 高度（估算值） | 间距 |
  |------|------------|------|
  | 邮箱输入行 | ~96px | — |
  | 分隔线 | 1px | — |
  | 说明文字 | ~56px | ~24px 上 |
  | 主按钮 | ~112px | ~32px 上 |

#### 邮箱输入行
- 高：~96px（估算值）
- 左：信封图标（~48×48px，估算值），右对齐至输入文字区
- 图标与占位文字「你的邮箱」水平对齐，垂直居中
- 图标颜色：#FFB7C5（主粉色）
- 占位文字颜色：#B0A8B4（浅灰紫，估算值）
- 无独立边框（卡片内部，靠分隔线区分区域）

#### 分隔线
- 颜色：rgba(0,0,0,0.06)（估算值）
- 高：1px
- 宽：全卡片内容区宽

#### 说明文字
- 文本：「我们会向你的邮箱发送同时登录链接，不需要密码。」
- 字号：~28px（估算值）
- 颜色：#8A8898（次要文字灰，估算值）
- 行高：~1.5
- 水平对齐：左对齐（卡片内部）

#### 主按钮「发送登录链接」
- 宽：全卡片内容区宽（~800px，估算值）
- 高：~112px（估算值）
- 圆角：~56px（pill 形，估算值）
- 背景：渐变（#FFB7C5 → #FF8FAB，左至右，估算值）
- 文字：「发送登录链接」
- 字号：~40px（估算值）
- 字重：Medium（500）
- 文字颜色：#FFFFFF

### Legal Zone（协议区）
- 宽：1024px（全宽）
- 高：~80px（估算值）
- 水平对齐：居中
- 文本：「继续即代表同意《用户协议》与《隐私政策》」
- 字号：~26px（估算值）
- 颜色：基础文字 #8A8898，链接部分《用户协议》《隐私政策》使用 #FFB7C5（主粉色）并加下划线

### Redeem Zone（兑换码区）
- 宽：1024px（全宽）
- 高：~96px（估算值）
- 水平对齐：居中
- 文本：「我有兑换码，直接激活 →」
- 字号：~30px（估算值）
- 颜色：#3A3A4A（墨色）或略浅
- 「→」箭头与文字同色，间距 ~8px

### Bottom Safe Area（底部安全区）
- 高：~96px（估算值，对应 iPhone Home Indicator 区域）
- 背景：与页面底部背景色一致（奶油粉白）

## 全局 Padding / Alignment

| 属性 | 值 |
|------|----|
| 页面左右外边距 | ~64px（估算值） |
| 表单卡左右偏移 | ~64px（与页面边距一致，估算值） |
| 各区块垂直间距 | ~32–48px（估算值） |
| 内部元素对齐 | 水平居中 / 文本左对齐（表单内） |

## 滚动区域说明
- 本页内容高度适配 iPhone 14 屏幕，正常情况下无需滚动
- 键盘弹起时，表单卡应上移，避免被键盘遮挡（需实现 KeyboardAvoidingView 或等效逻辑）
- 键盘弹起时，插画区可适度压缩或隐藏
