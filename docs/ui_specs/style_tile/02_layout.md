# 02 布局 — Style Tile（视觉总谱）

> 本文件描述的是参考画布的布局结构，非 App 可交互屏幕布局。
> 画布尺寸：**竖版**，PNG 实际像素约 **840×1536**（基于视觉判断，原始规格为 1024×1536 横版，但图像呈现为竖向滚动画布）。

---

## 整体画布结构

```
┌─────────────────────────────────────────┐  ← 画布顶部
│  Logo 区 "yuoyuo"（左上角）              │  H ≈ 80px（估算值）
│  背景：暖粉渐变 + 樱花装饰               │
├──────────────────┬──────────────────────┤
│  1. 调色板 卡片   │  2. 字体 卡片        │  Row-1, H ≈ 200px（估算值）
│  （左列）        │  （右列）             │
├──────────────────┼──────────────────────┤
│  3. 气泡 卡片    │  4. 按钮 卡片        │  Row-2, H ≈ 280px（估算值）
│  （左列）        │  （右列）             │
├──────────────────┴──────────────────────┤
│  5. 卡片 区块    │  6. 图标条 区块       │  Row-3, H ≈ 220px（估算值）
│  （左列）        │  （右列）             │
├─────────────────────────────────────────┤
│  全宽底部插画纹理（城堡 + 花卉）         │  H ≈ 260px（估算值）
└─────────────────────────────────────────┘  ← 画布底部
```

## 层级 Z-index 定义

| 层级 | 元素 |
|------|------|
| z=0 | 画布背景渐变 |
| z=1 | 底部插画纹理 |
| z=5 | 卡片阴影层 |
| z=10 | 各卡片内容 |
| z=15 | Logo 文字 |
| z=20 | 背景装饰粒子（樱花、星点） |

---

## Flex / Grid 对齐原则（推断）

- 整体网格：`display: flex; flex-wrap: wrap; gap: 16px; padding: 24px`（估算值）
- 每行两列：`flex: 1 1 calc(50% - 8px)`（估算值）
- 卡片内部：`display: flex; flex-direction: column; gap: 12px; padding: 20px`（估算值）
- 颜色椭圆行：`display: flex; flex-direction: row; gap: 8px; align-items: center`（估算值）
- 按钮列：`display: flex; flex-direction: column; gap: 12px; align-items: center`（估算值）
- 图标行：`display: flex; flex-direction: row; justify-content: space-around`（估算值）
