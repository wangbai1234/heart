# 06 交互规格 — Style Tile（视觉总谱）

> Style Tile 是静态参考画布，本身无交互行为。
> 本文件基于 Style Tile 中展示的 UI 元素，推断其在实际 App 中的交互规格；
> 所有交互状态均为设计系统推断，明确标注「设计稿未定义」的需后续单独确认。

---

## 通用说明

本文档区分两种标注：
- **设计稿已呈现**：PNG 中可视化呈现了该状态（如按钮默认态、气泡布局）
- **设计稿未定义**：PNG 未展示该交互态，属于设计系统推断或占位说明

---

## 1. 页面进入 / 退出动画

**N/A for this asset type**

Style Tile 为静态文档画布，不存在页面级进入/退出行为。各 App 屏幕的页面转场规格见对应屏幕规格文档。

参考值（设计系统推断）：
- 进入：`fade-in + slide-up`，duration: 300ms，easing: `easing-decelerate`
- 退出：`fade-out + slide-down`，duration: 250ms，easing: `easing-accelerate`

## 2. 按钮交互状态

### 2.1 Primary 按钮（"确认"）

| 状态 | 视觉表现 | 说明 |
|------|----------|------|
| **Default** | 樱花粉渐变背景 + 粉色外发光 | 设计稿已呈现 |
| **Hover（Web）** | 背景亮度提升约 5%，光晕扩散 | 设计稿未定义 |
| **Pressed / Active** | 背景加深约 10%（`#FF9BAE`），scale: 0.97，光晕收缩 | 设计稿未定义 |
| **Disabled** | opacity: 0.40，移除阴影 | 设计稿未定义 |
| **Loading** | 按钮内出现旋转 Spinner（白色，16px），文字隐藏 | 设计稿未定义 |

动效参数（设计系统推断）：
- Pressed 响应时长：150ms，`easing-standard`
- 回弹：spring(1, 100, 10, 0)，duration: 200ms

### 2.2 Secondary 按钮（"取消"）

| 状态 | 视觉表现 | 说明 |
|------|----------|------|
| **Default** | 白色背景 + 浅灰描边 | 设计稿已呈现 |
| **Pressed** | 背景变为 `rgba(0,0,0,0.04)`，scale: 0.97 | 设计稿未定义 |
| **Disabled** | opacity: 0.40 | 设计稿未定义 |

### 2.3 Text Link 按钮（"了解更多"）

| 状态 | 视觉表现 | 说明 |
|------|----------|------|
| **Default** | 薰衣草紫文字，无底色 | 设计稿已呈现 |
| **Pressed** | 文字颜色加深（`#A090E0`），opacity: 0.80 | 设计稿未定义 |
| **Visited** | 颜色变为 `color-ink-secondary` | 设计稿未定义 |

## 3. 气泡交互

### 3.1 AI 气泡

| 交互 | 行为 | 说明 |
|------|------|------|
| **入场动画** | 从左侧 fade-in + slide-right（translateX -8px → 0），duration: 250ms | 设计稿未定义 |
| **Long Press** | 弹出操作菜单（复制/收藏/删除），参考微信气泡菜单 | 设计稿未定义 |
| **Tap** | 无特殊反应（普通文字气泡） | 设计稿未定义 |
| **包含链接时** | 链接文字高亮为 `color-accent`，可点击 | 设计稿未定义 |

### 3.2 用户气泡

| 交互 | 行为 | 说明 |
|------|------|------|
| **入场动画** | 从右侧 fade-in + slide-left（translateX +8px → 0），duration: 250ms | 设计稿未定义 |
| **Long Press** | 同 AI 气泡操作菜单 | 设计稿未定义 |
| **已读状态变化** | 发送中 → 已发送（单勾）→ 已读（蓝色单勾）；可能有单勾→蓝勾的颜色过渡动画 | 设计稿未定义 |

## 9. Skeleton / Loading 状态

**设计稿未定义**

推断建议：
- 各卡片在数据加载前展示 Skeleton 骨架屏
- 骨架颜色：`rgba(0,0,0,0.06)` → `rgba(0,0,0,0.12)` 闪烁渐变（Shimmer 动效）
- Shimmer duration: 1200ms，loop

---

## 10. Error / Empty 状态

**设计稿未定义**

推断建议：
- 气泡发送失败：气泡右侧出现红色感叹号图标
- 网络断开：顶部出现提示 Banner（`color-error` 背景）
- 空消息列表：居中展示角色插画 + 提示文字（参考 `state_empty` 规格）
