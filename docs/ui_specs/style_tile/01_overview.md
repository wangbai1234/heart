# 01 概览 — Style Tile（视觉总谱）

---

## 作用与目标

Style Tile 是 yuoyuo 整套视觉语言的「一图通览基准」。它并非一个可交互屏幕，而是一张横版参考画布（1536×1024 像素），将颜色、字体、气泡、按钮、卡片、图标条、插画纹理等核心视觉元素集中呈现，供设计团队、工程师、产品经理在开发过程中随时对焦视觉标准。

**设计目标：**
1. 建立唯一真实来源（Single Source of Truth），避免各屏幕"各自发挥"。
2. 在正式开发前对齐全团队的美感认知与技术实现预期。
3. 输出可提取的 Design Token，直接服务于后续组件库搭建。

## 画布结构（视觉分区）

画布采用竖向卡片网格布局，6 个编号区块从上至下、从左至右排列：

| 编号 | 标题 | 内容摘要 |
|------|------|----------|
| 1 | 调色板 | 6 枚颜色椭圆 + Hex 标注 |
| 2 | 字体 | 中文大标题 / 正文 / 英文标题三个层级 |
| 3 | 气泡 | AI侧左气泡 + 用户侧右气泡 |
| 4 | 按钮 | Primary / Secondary / Text Link 三态 |
| 5 | 卡片 | AI 角色卡片（含头像、等级、统计数据、签名） |
| 6 | 图标条 | 底部导航 5 枚 Icon + 文字标签 |

画布底部有全宽梦幻插画（城堡、花卉、蝴蝶），作为背景纹理示意。

## 设计理念

yuoyuo 的视觉语言以「治愈系二次元陪伴」为核心叙事：

- **60% WeChat 清晰度**：信息层级清晰，气泡布局延续用户心智模型，降低认知门槛。
- **40% 原神温度**：借鉴《原神》的暖色渐变、动漫角色插画、光晕装饰，塑造情感温度。
- **Glassmorphism**：卡片与气泡大量使用白色半透明背景（rgba(255,255,255,0.55)），在暖奶色底层上浮现柔和光泽感。
- **樱花粉主色**：#FFB7C5 传递「亲密、温柔、少女」情绪，天蓝辅色 #A7C7E7 带来「平静、陪伴、可靠」感受，薰衣草紫 #C8B6FF 点缀「梦幻、奇思」气质。
