## 资源导出规格总结

| 资源 | 格式 | 分辨率 |
|------|------|--------|
| 角色头像插画 | PNG（含透明通道） | @1x / @2x / @3x |
| 底部场景插画 | WebP 优先 / PNG 备选 | @1x / @2x |
| Icon 组 | SVG（首选）+ PNG @3x | 24px → SVG |
| 装饰粒子 / 花瓣 | SVG（可程序化散布） | 矢量 |
| 心形 / 星星 Icon | SVG | 矢量 |
