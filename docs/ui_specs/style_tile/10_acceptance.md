## 验收通过标准

| 级别 | 标准 |
|------|------|
| **通过** | A–L 所有项目均为 ✅，或仅有 ≤3 项次要偏差（有设计师书面确认） |
| **条件通过** | 4–8 项有轻微偏差，设计师出具 Deviation 说明 |
| **不通过** | 任一 P0 项目（Logo 文字、Primary 按钮、气泡、角色卡片主要元素）不满足，或偏差项 > 8 项 |

**目标视觉一致度：≥ 95%**
