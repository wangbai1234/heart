## Z-index 层级（全局）

| 层 | 内容 | Z-index 估算 |
|----|------|-------------|
| 最底层 | 页面背景色 | 0 |
| 插画层 | 云朵光晕（渐变） | 1 |
| 内容层 | 心形宝石、文字、气泡 | 2 |
| Header 层 | 顶栏（静态，无模糊） | 10 |
| Input Bar 层 | 底部输入栏 | 10 |
| Shimmer 动画层 | 骨架高亮扫光 | 在内容层内（3） |
