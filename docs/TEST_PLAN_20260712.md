## 12. 出问题时怎么快速定位

| 症状 | 首查 | 常见根因 |
|---|---|---|
| 空气泡 / dots 永久卡住 | 后端 `_post_turn_billing` 日志、前端 `useWebSocket` watchdog | LLM 空返回 → PR #161 已发 `EMPTY_RESPONSE`；watchdog 60s 兜底 |
| 语音是默认少女音 | `env` 里 `MINIMAX_RIN_CLONE_VOICE_ID` | 未设 / 服务器未热启动 |
| 「」 拆气泡 | `heart/ss05_composer/message_splitter.py` corner_quote_depth | 服务器代码没到 53d8c8b |
| Clone "假成功" | `heart/api/routes_voice.py _call_tts_clone_api` | 服务器代码没到 845d25d（PR #161 之前的） |
| 已读不消 | `POST /api/chat/{cid}/mark-read` 是否被 400ms debounce 触发 | 前端 useEffect 依赖没变 → 强刷 |
| 头像 200000 报错 | `web/src/utils/imageCompress.ts` 有没有被调用 | 前端 dist 缓存 → SW 更新 |
